# استقرار پوشه Solution در برنامه‌های NET.
<hr className="mb-2" />
در برنامه‌های دات‌نت، دایرکتوری solution یک پوشه است که شامل یک فایل <Important>sln.</Important> و زیرپوشه‌های پروژه‌های مختلف است. فایل  <Important>sln.</Important> نقش مدیریت و هماهنگی بین پروژه‌های مختلف را دارد.


<div className="h-2" />
