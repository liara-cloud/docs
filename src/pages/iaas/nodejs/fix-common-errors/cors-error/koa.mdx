# رفع خطای CORS در فریم‌ورک Koa
<hr className="mb-2" />

برای رفع این خطا در فریم‌ورک Koa در ابتدا باید ماژول‌ <Important>cors</Important> را با اجرای دستور زیر، نصب کنید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="bash">
        {`npm install @koa/cors`}
    </Highlight>
</div>
<div className="h-2" />

در نهایت، می‌توانید در برنامه خود، مانند قطعه کد زیر، عمل کنید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`const Koa = require('koa');
const cors = require('@koa/cors');
const app = new Koa();

// Enable CORS for all origins
app.use(cors());

app.listen(3000, () => {
  console.log('Server is running on port 3000');
});`}
    </Highlight>
</div>
<div className="h-2" />
