# راه اندازی استک LAMP
<hr className="mb-2" />

استک LAMP شامل Linux (سیستم‌عامل)، Apache (وب سرور)، MySQL (پایگاه داده) و PHP (مفسر سمت سرور) است. این استک یکی از محبوب‌ترین پیکربندی‌ها برای میزبانی وبسایت‌های پویا محسوب می‌شود.
