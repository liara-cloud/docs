# استقرار برنامه‌های Svelte Kit در لیارا
<hr className="mb-2" />
<a href="https://qwik.dev/" className="text-[#2196f3] ">Qwik</a> یک فریم‌ورک جدید در جاوااسکریپت است که توسط تیم سازنده‌ی Builder.io توسعه یافته است. این فریم‌ورک طراحی شده است تا برنامه‌های وب سریع و بهینه‌ای را با تمرکز بر عملکرد بالا و زمان بارگذاری کم ایجاد کند. 
شما می‌توانید برنامه‌های Qwik خود را با <a href="../../how-tos/create-app" className="text-[#2196f3] ">ایجاد برنامه‌های NodeJS</a> در لیارا، مستقر کنید.
<div className='h-2' />

توجه داشته باشید که برای استقرار برنامه‌های Qwik تنها کافیست تا در مسیر اصلی پروژه، دستور زیر را اجرا کنید:


<div className="h-2" />
<div dir='ltr'>
    <Highlight className="bash">
        {`npm run qwik add`}
    </Highlight>
</div>
<div className="h-2" />

بعد از اجرای این دستور، بایستی گزینه <Important>Adapter: Node.js Server</Important> را انتخاب کرده و سپس گزینه <Important>!Yes looks good, finish update</Important> را بزنید.


<div className="h-4" />
<img src="https://files.liara.ir/liara/docs/qwik-create-deployment-files.gif" alt="work with qwik" />
<div className="h-4" />
با اجرای دستور فوق، یک‌سری فایل جدید برای شما ایجاد می‌شوند. اکنون بایستی اسکریپت <Important>start</Important> را به شکل زیر تغییر دهید:

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="json">
        {`"scripts": {
    // other scripts
    "start": "node server/entry.node-server",
    // other scripts
},`}
    </Highlight>
</div>
<div className="h-2" />
