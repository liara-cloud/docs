# استفاده از متغیرهای محیطی 
<hr className="mb-2" />
برای استفاده از متغیرهای محیطی در برنامه خود، در ابتدا باید طبق <Link href="/paas/details/envs" className="text-[#2196f3] ">مستندات تنظیم متغیرهای محیطی</Link>، متغیرهای محیطی را به برنامه خود، اضافه کنید.
در ادامه، شما می‌توانید با استفاده از ماژول‌ os و متد <Important>()Getenv</Important> به متغیرهای محیطی خود در برنامه go دسترسی داشته باشید؛ به عنوان مثال:

<div className="h-2" />
<div dir="ltr">
<Highlight  className="go">
{`package main

import (
	"fmt"
	"os"
)

func main() {
	value := os.Getenv("LIARA_URL")
	if value == "" {
		fmt.Println("LIARA_URL is not set")
	} else {
		fmt.Println("LIARA_URL:", value)
	}
}
`}
</Highlight>
</ div>
<div className="h-2" />
در مثال فوق، <Important>LIARA_URL</Important> یک متغیر محیطی است که از قبل، به برنامه فرضی در لیارا، اضافه شده است.
