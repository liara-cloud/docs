# شروع به کار با پلتفرم React
<hr className="mb-2" />

<a href="https://react.dev/" className="text-[#2196f3] ">React</a> یک فریم‌ورک جاوااسکریپت برای ساخت رابط‌های کاربری است که توسط FaceBook توسعه داده شده است. این فریم‌ورک مبتنی بر کامپوننت‌هاست و هر کامپوننت می‌تواند به صورت مستقل توسعه و استفاده شود.
<div className="h-2" />

React از یک ساختار مجازی به نام Virtual DOM استفاده می‌کند که به بهبود عملکرد و کارایی برنامه‌ها کمک می‌کند. با استفاده از JSX، می‌توان کدهای HTML را به طور مستقیم در جاوااسکریپت نوشت. این فریم‌ورک به خصوص برای ساخت برنامه‌های تک‌صفحه‌ای (SPA) مناسب است.
شما می‌توانید در سریع‌ترین زمان ممکن و به سادگی چند کلیک یا اجرای چند دستور ساده، برنامه خود را در لیارا مستقر کنید. در ادامه بخوانید:
