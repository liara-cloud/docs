# فعال‌سازی SSR با استفاده از Inertia در Laravel
<hr className="mb-2" />

SSR یا Server-Side Rendering به فرآیند رندرینگ صفحات وب در سمت سرور، قبل از ارسال به مرورگر کاربر، گفته می‌شود. از جمله مزایای این قابلیت، بارگذاری سریع‌تر و SEO بهتر برنامه، است.
<a href="https://inertiajs.com/" className="text-[#2196f3] ">InertiaJS</a> یک کتابخانه است که به توسعه‌دهندگان اجازه می‌دهد تا با استفاده از فریم‌ورک‌های سمت سرور مانند Laravel و فریم‌ورک‌های جاوااسکریپت سمت کلاینت مانند <a href="/paas/vue/getting-started" className="text-[#2196f3] ">Vue</a> یا <a href="/paas/react/getting-started" className="text-[#2196f3] ">React</a>، اپلیکیشن‌های SPA بسازند بدون اینکه نیاز به APIهای جداگانه باشد.
    <div className="h-2" />
