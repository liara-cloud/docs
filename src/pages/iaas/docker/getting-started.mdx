# شروع به کار با پلتفرم Docker
<hr className="mb-2" />

<a href="https://www.docker.com/" className="text-[#2196f3] ">Docker</a> یک پلتفرم متن‌باز است که برای خودکارسازی استقرار برنامه‌ها در کانتینرها استفاده می‌شود. کانتینرها محیط‌های سبک، قابل حمل و ایزوله‌ای هستند که شامل همه اجزای لازم برای اجرای یک نرم‌افزار هستند. Docker به توسعه‌دهندگان اجازه می‌دهد تا برنامه‌ها و وابستگی‌هایشان را بسته‌بندی کرده و بدون توجه به محیط میزبان، به راحتی اجرا کنند. این فناوری باعث افزایش کارایی، تسهیل در توسعه و تست نرم‌افزار و بهبود قابلیت انتقال بین محیط‌های مختلف می‌شود.
شما می‌توانید در سریع‌ترین زمان ممکن و به سادگی چند کلیک یا اجرای چند دستور ساده، برنامه داکرایز شده خود را در لیارا مستقر کنید. در ادامه بخوانید:
