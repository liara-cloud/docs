# استقرار برنامه‌های Bun در لیارا
<hr className="mb-2" />
<a href="https://bun.sh/" className="text-[#2196f3] ">Bun</a> یک محیط اجرای جاوااسکریپت/تایپ‌اسکریپت و یک باندلر (bundler) سریع است که برای توسعه وب طراحی شده. این ابزار به‌طور خاص برای سرعت بالا و کارایی بهتر در مقایسه با ابزارهای مشابه مثل Webpack و Vite ساخته شده است. Bun از Rust برای بهینه‌سازی عملکرد استفاده می‌کند و شامل ابزارهایی مانند یک HTTP server، یک task runner و یک bunder می‌شود. هدف Bun این است که توسعه‌دهندگان را قادر سازد تا برنامه‌های وب را سریع‌تر از همیشه بسازند و اجرا کنند.

<div className="h-2" />

شما می‌توانید برنامه‌های Bun خود را با <a href="../../how-tos/create-app" className="text-[#2196f3] ">ایجاد برنامه‌های Docker</a> در لیارا، مستقر کنید.
برای این‌کار، کافیست تا در مسیر اصلی پروژه در Local، یک فایل به نام <Important>Dockerfile</Important> ایجاد کنید و قطعه کد زیر را درون آن، قرار دهید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="dockerfile">
        {`# Step 1: Use the official Bun image
FROM oven/bun:1.1.37

# Step 2: Set the working directory in the container
WORKDIR /app

# Step 3: Copy the application files into the container
COPY . .

# Step 4: Install Bun dependencies
RUN bun install

# Step 5: Run the application
CMD ["bun", "server.ts"]
`}
    </Highlight>
</div>
<div className="h-2" />

در نهایت، در مسیری که <Important>Dockerfile</Important> قرار گرفته است، دستور زیر را اجرا کنید تا برنامه‌تان در لیارا مستقر شود:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`liara deploy --platform=docker --port=3000`}
    </Highlight>
</div>
<div className="h-2" />
البته اگر که برنامه‌تان در پورت دیگری به جز 3000 به درخواست کاربران، گوش می‌دهد؛ بایستی به جای 3000، مقدار پورت خود را، وارد کنید.
