# آیدی یک سرور مجازی ابری

<hr className="mb-2" />

آیدی سرور مجازی (vmID) یک شناسه منحصربه‌فرد است که لیارا به هر سرور مجازی ابری که ایجاد می‌کنید، اختصاص می‌دهد. این آیدی برای مشاهده رویدادهای سرور مجازی و ارسال سیگنال‌های قطع برق، روشن کردن، خاموش کردن و ری‌استارت از طریق API به کار می‌رود. هنگام ارسال درخواست به API لیارا، باید این مقدار را در URL درخواست قرار دهید تا سرور مجازی ابری شما، مشخص شود.


<Section id="how-to-see-vm-id" title="نحوه مشاهده آیدی سرور مجازی ابری" />
روش‌های مختلفی برای مشاهده آیدی سرور مجازی مدنظرتان، وجود دارد 
که یکی از آن‌ها، استفاده از URL کنسول لیارا در حین مشاهده سرور مجازی است.

<div className="h-4" />
<img src="https://media.liara.ir/vps/see-vmid-in-console.png" alt="work with panel"/>
<div className="h-4" />
آیدی سرور مجازی بعد از <Important>/vms</Important> و قبل از <Important>/</Important> بعدی، قرار دارد.
به‌عنوان مثال، اگر URL کنسول شما در صفحه اطلاعات کلی سرور مجازی‌تان مانند زیر باشد:

<div dir='ltr'>

        {`https://console.liara.ir/vms/67cfe08f8ff9796bd0d7045f/overview`}
    </Highlight>
</div>
<div className="h-2" />

آیدی سرور مجازی شما برابر با، <Important>67cfe08f8ff9796bd0d7045f</Important> است.
