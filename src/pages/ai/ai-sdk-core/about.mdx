# آشنایی با قابلیت‌های AI SDK
<hr className="mb-2" />

مدل‌های زبانی بزرگ (Large Language Modelها یا LLMها) برنامه‌های پیشرفته‌ای هستند که قادرند زبان انسانی را در مقیاس وسیع درک و تولید کرده و با آن تعامل برقرار کنند. این مدل‌ها با استفاده از حجم عظیمی از متون، آموزش داده می‌شوند تا الگوهای موجود در زبان را تشخیص داده و پیش‌بینی کنند که در یک قطعه‌ی متنی، کلمه یا عبارت بعدی چه خواهد بود.
<div className="h-4" />


هسته‌ی AI SDK، کار با LLMها را با فراهم‌کردن یک روش استاندارد برای یکپارچه‌سازی آن‌ها در اپلیکیشن شما ساده‌تر می‌کند؛ به این ترتیب، می‌توانید تمرکز خود را بر ساخت برنامه‌های هوش مصنوعی برای کاربران‌تان بگذارید و نه برای جزئیات فنی.
<div className="h-4" />

هسته‌ی AI SDK شامل مجموعه‌ای از توابع است که برای تولید متن، تولید داده‌ی ساختارمند و استفاده از Toolها طراحی شده‌اند. این توابع از یک روش استاندارد برای تنظیم پرامپت‌ها (prompts) و پیکربندی‌ها استفاده می‌کنند تا کار با مدل‌های مختلف را ساده‌تر کنند.

<div className='h-4' />
<ul>
<li>تابع <Important>generateText</Important>: این تابع متن و tool call تولید می‌کند و برای موارد استفاده غیرتعاملی نظیر اتوماسیون تسک‌ها (جایی که شما نیاز به نوشتن متن دارید، مانند پیش‌نویس ایمیل یا خلاصه‌سازی صفحات وب) و برای agentهایی که از toolها استفاده می‌کنند، ایده‌آل است. </li>
<div className='h-4' />

<li>تابع <Important>streamText</Important>: این تابع متن و tool callها را استریم می‌کند. شما می‌توانید از این تابع، برای موارد استفاده تعاملی نظیر چت‌بات‌ها و استریم محتوا، استفاده کنید</li>
<div className='h-4' />

<li>تابع <Important>generateObject</Important>: این تابع، یک آبجکت دارای ساختار و تایپ مشخص، تولید می‌کند که با اسکیمای Zod، همخوانی دارد. شما می‌‌توانید از این تابع استفاده کنید تا مدل را مجبور کنید که داده‌های ساختارمند را return کند</li>
<div className='h-4' />

<li>تابع <Important>streamObject</Important>: این تابع، آبجکت دارای ساختاری که با اسکیمای Zod همخوانی دارد را، استریم می‌کند.</li>

</ul>
