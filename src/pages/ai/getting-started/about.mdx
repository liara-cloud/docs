# آشنایی با AI SDK
<hr className="mb-2" />

<a href="https://ai-sdk.dev/" className="text-[#2196f3]">AI SDK</a> یک کیت توسعه نرم‌افزار (Software Development Kit) است که برای تعامل ساده، سریع و انعطاف‌پذیر با مدل‌های هوش مصنوعی در JavaScript و TypeScript طراحی شده است. این SDK یک رابط واحد و استاندارد ارائه می‌دهد که به شما اجازه می‌دهد بدون وابستگی به ارائه‌دهنده‌ی خاص، مدل‌های مختلف زبانی را فراخوانی و مدیریت کنید. از مزیت‌های اصلی آن می‌توان به پشتیبانی از چندین ارائه‌دهنده (مثل OpenAI و Anthropic و HuggingFace)، مدیریت هوشمند توکن‌ها و درخواست‌ها، و امکانات پیشرفته برای کنترل ورودی/خروجی اشاره کرد.
<div className="h-4" />

سرویس هوش مصنوعی لیارا، از جمله سرویس‌هایی است که با AI SDK به‌طور کامل سازگار است و شما می‌توانید با ترکیب هوش مصنوعی لیارا و AI SDK، برنامه‌های قدرتمند و در عین حال، به‌صرفه، در زمینه هوش مصنوعی، ایجاد کنید.
