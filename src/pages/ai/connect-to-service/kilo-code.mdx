# اتصال هوش مصنوعی به افزونه Kilo Code
<hr className="mb-2" />

افزونه <a href="https://kilocode.ai" className="text-[#2196f3]">Kilo Code</a> یک ابزار توسعه برای محیط برنامه‌نویسی VS Code است که با هدف تسهیل و تسریع فرآیند کدنویسی طراحی شده است. این افزونه با استفاده از قابلیت‌های هوش مصنوعی، به برنامه‌نویسان کمک می‌کند تا کدهایی با کیفیت بالاتر و در زمان کمتر تولید کنند. از جمله قابلیت‌های آن می‌توان به تکمیل خودکار کد، پیشنهادهای هوشمند، تولید قطعه کدهای آماده (code snippets) و امکان تولید کد بر اساس توضیحات متنی اشاره کرد. Kilo Code برای توسعه‌دهندگان مبتدی و کسانی که زمان محدودی برای نوشتن کد دارند، بسیار مفید است.

<div className="h-2" />
