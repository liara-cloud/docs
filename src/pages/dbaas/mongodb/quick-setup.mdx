# راه‌اندازی سریع دیتابیس MongoDB
<hr className="mb-2" />

<a href="https://www.mongodb.com/" className="text-[#2196f3] ">MongoDB</a> یک پایگاه داده NoSQL مبتنی بر document است که داده‌ها را به صورت مستندات JSON ذخیره می‌کند. برخلاف پایگاه‌های داده رابطه‌ای، MongoDB از جداول و سطرها استفاده نمی‌کند و به جای آن از ساختارهای منعطف‌تر و مقیاس‌پذیرتری استفاده می‌کند که به راحتی قابلیت تغییر و توسعه دارند. این ویژگی‌ها باعث می‌شود که MongoDB برای مدیریت داده‌های غیرساختاریافته و کاربردهای بزرگ و پیچیده مناسب باشد.
