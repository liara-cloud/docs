# اتصال به دیتابیس MongoDB با mongosh
<hr className="mb-2" />

<a href="https://www.mongodb.com/docs/mongodb-shell/" className="text-[#2196f3] ">mongosh</a>، که مخفف MongoDB Shell است، یک رابط خط فرمان جدید برای کار با MongoDB است. این ابزار برای تعامل با پایگاه‌داده MongoDB استفاده می‌شود و قابلیت اجرای دستورات JavaScript را نیز دارد. mongosh جایگزین ابزار قدیمی mongo shell شده است و امکانات بیشتری برای توسعه‌دهندگان فراهم می‌کند، از جمله تجربه کاربری بهتر، ویژگی‌های تکمیل خودکار و پشتیبانی از انواع داده‌های پیشرفته. همچنین، این ابزار به‌راحتی با انواع مختلف سیستم‌عامل‌ها سازگار است.

<div className="h-2" />

برای اتصال به mongosh، پس از <a href="https://www.mongodb.com/docs/mongodb-shell/install/" className="text-[#2196f3] ">نصب آن طبق مستندات رسمی</a> کافیست از اطلاعات شبکه عمومی دیتابیس، که در صفحه <b>نحوه اتصال</b> موجود است؛ استفاده کنید.
برای فعال‌سازی شبکه عمومی نیز، می‌توانید از قسمت <b>تنظیمات دیتابیس</b>، این کار را انجام دهید. 
برای اتصال، می‌توانید مشابه دستور زیر نیز، عمل کنید:

<div className="h-4" />
    <div dir='ltr'>
    <Highlight className="bash">
        {`mongosh -u DB_USERNAME \\
           --port DB_PORT \\
           --host DB_HOST \\
           -p DB_PASSWORD \\
           --authenticationDatabase admin`}
    </Highlight>
    </div>
