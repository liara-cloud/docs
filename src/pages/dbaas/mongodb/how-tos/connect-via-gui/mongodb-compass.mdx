# اتصال به دیتابیس MongoDB در MongoDB Compass
<hr className="mb-2" />

<a href="https://www.mongodb.com/products/tools/compass" className="text-[#2196f3] ">MongoDB Compass</a> یک ابزار برای MongoDB است که به کاربران امکان می‌دهد به راحتی با پایگاه داده‌های MongoDB کار کنند. با استفاده از این ابزار، کاربران می‌توانند داده‌ها را مشاهده، تحلیل، و مدیریت کنند، همچنین کوئری‌های پیچیده را بدون نیاز به نوشتن کد اجرا کنند. MongoDB Compass قابلیت‌های متعددی از جمله نمایش ساختار داده‌ها، شاخص‌ها، و ابزارهای تجزیه و تحلیل عملکرد را فراهم می‌کند که به بهینه‌سازی و مدیریت بهتر پایگاه داده‌ها کمک می‌کند.

<div className="h-2" />

برای اتصال به MongoDB Compass کافیست از اطلاعات شبکه عمومی دیتابیس، که در صفحه <b>نحوه اتصال</b> موجود است؛ استفاده کنید.
برای فعال‌سازی شبکه عمومی نیز، می‌توانید از قسمت <b>تنظیمات دیتابیس</b>، این کار را انجام دهید.
