# اتصال به دیتابیس Redis با redis-cli
<hr className="mb-2" />

<a href="https://redis.io/docs/latest/develop/connect/cli/" className="text-[#2196f3] ">redis-cli</a> یک ابزار خط فرمان برای تعامل با پایگاه داده Redis است. با استفاده از redis-cli می‌توان به سرور Redis متصل شد، دستورات مختلفی مانند SET , GET , DEL و دیگر عملیات‌ها را اجرا کرد، اطلاعات مربوط به پایگاه داده را بررسی کرد و خطاها را دیباگ نمود. این ابزار برای توسعه‌دهندگان و مدیران سیستم‌ها بسیار مفید است تا بتوانند به راحتی با Redis کار کنند و مدیریت و پیکربندی‌های لازم را انجام دهند.

<div className="h-2" />

برای اتصال به redis-cli، پس از <a href="https://redis.io/docs/latest/operate/oss_and_stack/install/install-redis/" className="text-[#2196f3] ">نصب آن طبق مستندات رسمی</a> کافیست از اطلاعات شبکه عمومی دیتابیس، که در صفحه <b>نحوه اتصال</b> موجود است؛ استفاده کنید.
برای فعال‌سازی شبکه عمومی نیز، می‌توانید از قسمت <b>تنظیمات دیتابیس</b>، این کار را انجام دهید. 
برای اتصال، می‌توانید مشابه دستور زیر نیز، عمل کنید:

<div className="h-4" />
    <div dir='ltr'>
    <Highlight className="bash">
        {`redis-cli -h REDIS_HOST -p REDIS_PORT -a REDIS_PASSWORD`}
    </Highlight>
    </div>

<Asciinema id="connect-redis" />

پس از اتصال به دیتابیس، می‌توانید عملیات مورد نیاز خود را، انجام دهید.
