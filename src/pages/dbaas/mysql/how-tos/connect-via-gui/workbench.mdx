# اتصال به دیتابیس MySQL در MySQL Workbench
<hr className="mb-2" />

<a href="https://www.mysql.com/products/workbench/" className="text-[#2196f3] ">MySQL Workbench</a> یک ابزار گرافیکی برای مدیریت و طراحی پایگاه‌داده‌های MySQL است. این برنامه امکاناتی مانند طراحی مدل‌های ER (مدل‌های موجودیت-رابطه)، ویرایش و مدیریت جداول و داده‌ها، اجرای کوئری‌ها و تحلیل عملکرد پایگاه‌داده را فراهم می‌آورد. MySQL Workbench به توسعه‌دهندگان و مدیران پایگاه‌داده کمک می‌کند تا فرآیند طراحی و نگهداری پایگاه‌داده‌ها را به‌صورت بصری و کارآمد انجام دهند.

<div className="h-2" />

برای اتصال به MySQL Workbench کافیست از اطلاعات شبکه عمومی دیتابیس، که در صفحه <b>نحوه اتصال</b> موجود است؛ استفاده کنید.
برای فعال‌سازی شبکه عمومی نیز، می‌توانید از قسمت <b>تنظیمات دیتابیس</b>، این کار را انجام دهید.
