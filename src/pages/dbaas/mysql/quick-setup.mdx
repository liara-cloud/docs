# راه‌اندازی سریع دیتابیس MySQL
<hr className="mb-2" />

<a href="https://www.mysql.com/" className="text-[#2196f3] ">MySQL</a> یک سیستم مدیریت پایگاه داده‌ی رابطه‌ای (RDBMS) است که توسط شرکت Oracle توسعه و پشتیبانی می‌شود. این نرم‌افزار از زبان SQL برای مدیریت و تعامل با داده‌ها استفاده می‌کند و به‌طور گسترده در برنامه‌های وب و سرورهای بزرگ به‌کار می‌رود. MySQL به دلیل عملکرد بالا، مقیاس‌پذیری و قابلیت‌های امنیتی قوی، انتخاب محبوبی در میان توسعه‌دهندگان است. این سیستم می‌تواند بر روی سیستم‌عامل‌های مختلفی از جمله ویندوز، لینوکس و مک اجرا شود.
