# اتصال به دیتابیس MariaDB با MariaDB-CLI
<hr className="mb-2" />

<a href="https://mariadb.com/kb/en/mariadb-command-line-client/" className="text-[#2196f3] ">MariaDB CLI</a> ابزاری است که برای تعامل با پایگاه داده‌های MariaDB از طریق خط فرمان استفاده می‌شود. این ابزار به کاربران اجازه می‌دهد تا دستورات SQL را به صورت مستقیم اجرا کرده و به مدیریت و مانیتورینگ پایگاه داده بپردازند. با استفاده از mariadb cli می‌توان عملیات‌هایی مانند ایجاد و حذف جداول، وارد کردن و استخراج داده‌ها، و اجرای پرس و جوها را انجام داد. این ابزار یکی از ابزارهای اصلی برای مدیریت MariaDB به شمار می‌آید و اغلب در محیط‌های توسعه و تولید استفاده می‌شود.

<div className="h-2" />

برای اتصال به MariaDB-CLI، پس از <a href="https://mariadb.com/docs/server/connect/clients/mariadb-client/#Installation" className="text-[#2196f3] ">نصب آن طبق مستندات رسمی</a> کافیست از اطلاعات شبکه عمومی دیتابیس، که در صفحه <b>نحوه اتصال</b> موجود است؛ استفاده کنید.
برای فعال‌سازی شبکه عمومی نیز، می‌توانید از قسمت <b>تنظیمات دیتابیس</b>، این کار را انجام دهید. 
برای اتصال، می‌توانید مشابه دستور زیر نیز، عمل کنید:

<div className="h-4" />
    <div dir='ltr'>
    <Highlight className="bash">
        {`mariadb -h <hostname> -u <username> -p<password> -P <port> -D <database_name>`}
    </Highlight>
    </div>
