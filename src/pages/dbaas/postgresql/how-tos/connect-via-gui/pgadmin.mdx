# اتصال به دیتابیس PostgreSQL در PGAdmin
<hr className="mb-2" />

<a href="https://www.pgadmin.org/" className="text-[#2196f3] ">PGAdmin</a> یک ابزار گرافیکی برای مدیریت پایگاه‌های داده PostgreSQL است. با استفاده از pgAdmin می‌توانید پایگاه‌های داده را ایجاد و مدیریت کنید، جداول و داده‌ها را ویرایش کنید، و پرس و جوهای SQL را اجرا کنید. این ابزار قابلیت‌هایی مانند پایش عملکرد، تنظیمات پیکربندی، و ایجاد گزارش‌های مختلف را نیز فراهم می‌کند. pgAdmin به عنوان یک رابط کاربری بصری برای تسهیل کار با PostgreSQL طراحی شده است.

<div className="h-2" />

برای اتصال به PGAdmin کافیست تا گزینه آن را در منوی <b>نحوه اتصال</b> دیتابیس خود، فعال کنید
برای ورود به حساب کاربری، می‌توانید از نام کاربری <Important>root@pgadmin.org</Important> و رمزعبور دیتابیس خود، استفاده کنید.
