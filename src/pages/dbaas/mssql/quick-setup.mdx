# راه‌اندازی سریع دیتابیس MSSQL
<hr className="mb-2" />

<a href="https://www.microsoft.com/en-us/sql-server/sql-server-downloads" className="text-[#2196f3] ">MSSQL</a> یک سیستم مدیریت پایگاه داده رابطه‌ای (RDBMS) توسعه‌یافته توسط مایکروسافت است. این نرم‌افزار برای ذخیره‌سازی، بازیابی و مدیریت داده‌ها در محیط‌های مختلف از جمله سرورها و دسکتاپ‌ها طراحی شده است. MSSQL از زبان SQL (Structured Query Language) برای مدیریت داده‌ها استفاده می‌کند و امکاناتی نظیر امنیت پیشرفته، قابلیت‌های پشتیبان‌گیری و بازیابی، و ابزارهای تحلیلی را ارائه می‌دهد. این سیستم به ویژه در سازمان‌های بزرگ و پروژه‌های پیچیده کاربرد دارد.
