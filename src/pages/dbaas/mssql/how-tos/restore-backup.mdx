# بازیابی فایل پشتیبان در دیتابیس
<hr className="mb-2" />

برای بازیابی فایل پشتیبان در دیتابیس‌های SQL Server می‌توانید وارد صفحه‌ی <b>پشتیبان‌گیری</b> دیتابیس خود شده و با کلیک روی گزینه‌ی <b>بازیابی فایل bak.</b> آدرس مستقیم فایل پشتیبان خود را وارد کنید.
شما می‌توانید آدرس مستقیم فایل پشتیبان را پس از آپلود در فضای ابری، در این قسمت وارد کنید. لیارا به‌صورت خودکار فایل پشتیبان را در مسیر <Important>var/opt/mssql/data/</Important> قرار می‌دهد و شما می‌توانید با استفاده از ابزار <a href="/dbaas/mssql/how-tos/connect-via-gui/mssql-server-studio" className="text-[#2196f3] ">SSMS</a>، فایل پشتیبان را از این مسیر در دیتابیس خود بازیابی کنید.
