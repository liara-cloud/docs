# اتصال به دیتابیس MSSQL در Azure Data Studio
<hr className="mb-2" />

<a href="https://azure.microsoft.com/en-us/products/data-studio" className="text-[#2196f3] ">Azure Data Studio</a> یک ابزار رایگان و متن‌باز برای مدیریت و تجزیه و تحلیل پایگاه‌های داده است که به ویژه برای کار با پایگاه‌های داده SQL Server و Azure SQL Database طراحی شده است. این محیط شامل ویژگی‌های مفیدی مانند ویرایشگر کوئری با قابلیت تکمیل خودکار، داشبوردهای تعاملی، و ابزارهای تجزیه و تحلیل داده‌ها است. همچنین از افزونه‌های متنوعی پشتیبانی می‌کند که قابلیت‌های آن را بهبود می‌بخشد و به کاربران اجازه می‌دهد تا با زبان‌های برنامه‌نویسی مختلف، از جمله Python و PowerShell، کار کنند.

<div className="h-2" />

برای اتصال به Azure Data Studio کافیست از اطلاعات شبکه عمومی دیتابیس، که در صفحه <b>نحوه اتصال</b> موجود است؛ استفاده کنید.
برای فعال‌سازی شبکه عمومی نیز، می‌توانید از قسمت <b>تنظیمات دیتابیس</b>، این کار را انجام دهید.
