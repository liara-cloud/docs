# استقرار برنامه‌های GatsbyJS در لیارا
<hr className="mb-2" />
<a href="https://www.gatsbyjs.com/" className="text-[#2196f3] ">GatsbyJS</a> یک فریم‌ورک متن‌باز مبتنی بر React است که برای ساخت وب‌سایت‌ها و برنامه‌های وب سریع و بهینه استفاده می‌شود. این فریم‌ورک با استفاده از GraphQL به توسعه‌دهندگان اجازه می‌دهد تا داده‌ها را از منابع مختلف (مثل CMSها، فایل‌های Markdown و APIها) دریافت و مدیریت کنند. GatsbyJS به دلیل تولید صفحات ایستا در زمان ساخت (static site generation) و همچنین بهینه‌سازی برای SEO و عملکرد بالا، محبوبیت زیادی پیدا کرده است.
<div className="h-2" />

شما می‌توانید برنامه‌های Gatsby خود را با <a href="../../how-tos/create-app" className="text-[#2196f3] ">ایجاد برنامه‌های Static</a> در لیارا، مستقر کنید.
برای این‌کار، کافیست تا در مسیر اصلی پروژه در Local، دستور زیر را اجرا کنید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="json">
        {`npm run build`}
    </Highlight>
</div>
<div className="h-2" />

با اجرای دستور فوق، فرایند build گرفتن از برنامه‌ی شما شروع خواهد شد و درنهایت تمام فایل‌های استاتیک اعم از فایل‌های <Important>HTML/CSS/JS</Important> در پوشه‌ی <Important>public</Important> قرار داده می‌شوند.
اکنون، کافیست تا وارد پوشه public شوید و قطعه کد زیر را در آن، اجرا کنید:

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`liara deploy --platform=static`}
    </Highlight>
</div>
<div className="h-2" />
با اجرای دستور فوق، برنامه‌تان در لیارا، مستقر خواهد شد.
