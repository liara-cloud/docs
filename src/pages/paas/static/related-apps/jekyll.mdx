# استقرار برنامه‌های Jekyll در لیارا
<hr className="mb-2" />
<a href="https://jekyllrb.com/" className="text-[#2196f3] ">Jekyll</a> یک static site generator متن‌باز است که با زبان Ruby نوشته شده و برای ساخت وب‌سایت‌ها و وبلاگ‌های سریع و ساده استفاده می‌شود. این ابزار با استفاده از قالب‌های ساده مانند Markdown و Liquid، فایل‌های متنی را به صفحات HTML تبدیل می‌کند. Jekyll به طور خاص برای میزبانی با GitHub Pages طراحی شده و به توسعه‌دهندگان امکان می‌دهد تا به راحتی وب‌سایت‌های خود را در GitHub منتشر کنند. این فریم‌ورک برای توسعه‌دهندگانی که به دنبال یک ابزار ساده و قابل اعتماد برای ساخت وب‌سایت‌های ایستا هستند، مناسب است.

<div className="h-2" />

شما می‌توانید برنامه‌های Jekyll خود را با <a href="../../how-tos/create-app" className="text-[#2196f3] ">ایجاد برنامه‌های Static</a> در لیارا، مستقر کنید.
برای این‌کار، کافیست تا در مسیر اصلی پروژه در Local، دستور زیر را اجرا کنید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="json">
        {`jekyll build`}
    </Highlight>
</div>
<div className="h-2" />

با اجرای دستور فوق، فرایند build گرفتن از برنامه‌ی شما شروع خواهد شد و درنهایت تمام فایل‌های استاتیک در پوشه‌ی <Important>site_</Important> قرار داده می‌شوند.
اکنون، کافیست تا وارد پوشه site_ شوید و قطعه کد زیر را در آن، اجرا کنید:

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`liara deploy --platform=static`}
    </Highlight>
</div>
<div className="h-2" />
با اجرای دستور فوق، برنامه‌تان در لیارا، مستقر خواهد شد.
