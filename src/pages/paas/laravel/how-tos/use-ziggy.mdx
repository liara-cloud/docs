# استفاده از Ziggy در برنامه‌های Laravel
<hr className="mb-2" />

Ziggy یک پکیج لاراول است که به شما امکان می‌دهد تا به راحتی به مسیر (route)های تعریف‌شده در لاراول از طریق جاوااسکریپت دسترسی داشته باشید. این ابزار برای توسعه‌دهندگانی که از فریم‌ورک‌های جاوااسکریپت سمت کاربر مانند <a href="/paas/vue/getting-started" className="text-[#2196f3] ">Vue</a> یا <a href="/paas/react/getting-started" className="text-[#2196f3] ">React</a> استفاده می‌کنند، بسیار مفید است.
<div className='h-2' />

برای استفاده از Ziggy در لیارا تنها کافیست فایل مورد نظرتان را از مسیر <Important>vendor/tightenco/ziggy/dist</Important> در پوشه‌ی <Important>resources/js</Important> کپی کنید. برای مثال، درصورتی که فایل <Important>vue.m</Important> را کپی کرده باشید؛ می‌توانید به‌شکل زیر آن را در برنامه‌ی خود import کنید:
<div className='h-2' />
<div dir='ltr'>
    <Highlight className="config">
        {`import { ZiggyVue } from './vue.m';
`}        
    </Highlight>
</div>
