# استفاده از ماژول FFMPEG در Laravel
<hr className="mb-2" />

<a href="https://ffmpeg.org/" className="text-[#2196f3] ">FFMPEG</a> یک نرم‌افزار متن‌باز برای پردازش چندرسانه‌ای است که قابلیت ضبط، تبدیل، پخش ویدیوی زنده و پخش استریمی (streaming) را دارد. این ابزار بسیار قدرتمند است و برای انواع مختلفی از فرمت‌های صوتی و تصویری، قابلیت‌ها و توابع متنوعی ارائه می‌دهد. 

ماژول FFMPEG به‌صورت پیش‌فرض در برنامه‌های Laravel نصب است و همچنین متغیرهای محیطی <Important>FFMPEG_PATH</Important> و <Important>FFPROBE_PATH</Important> در این پلتفرم تنظیم شده‌اند. شما برای استفاده از این ماژول تنها کافیست پکیج <Important>php-ffmpeg</Important> را با اجرای دستور زیر نصب کنید:


<div className='h-2' />
<div dir='ltr'>
    <Highlight className="bash">
        {`composer require php-ffmpeg/php-ffmpeg`}        
    </Highlight>
</div>
<div className="h-2" />
در نهایت، می‌توانید همانند قطعه کد زیر، از این پکیج در برنامه خود، بهره ببرید:
<div className="h-2" />
<div dir='ltr'>
        <Highlight className="php">
            {`use FFMpeg;

$ffmpeg = FFMpeg::create([
    'ffmpeg.binaries' => env('FFMPEG_PATH', '/usr/bin/ffmpeg'),
    'ffprobe.binaries' => env('FFPROBE_PATH', '/usr/bin/ffprobe'),
]);`}
</Highlight>
</div>
