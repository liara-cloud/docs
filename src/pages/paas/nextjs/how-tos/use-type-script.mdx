# استفاده از TypeScript در برنامه‌های NextJS
<hr className="mb-2" />

برای استقرار برنامه‌های NextJS مبتنی بر TypeScript نیاز به انجام کار خاصی نیست و پلتفرم NextJS لیارا به صورت کامل 
از آن، پشتیبانی می‌کند. تنها نکته قابل توجه این است که فایل <Important>package.json</Important> برنامه‌تان باید در <Important>devDependencies</Important>شامل ماژول TypeScript باشد. به عنوان مثال:

<div className="h-2" />
<div dir="ltr">
<Highlight className='json'>
{`"devDependencies": {
    "typescript": "^5",
    "@types/node": "^20",
    "@types/react": "^18",
    "@types/react-dom": "^18",
    "eslint": "^8",
    "eslint-config-next": "14.2.3"
  }`}
</Highlight>
</div>
