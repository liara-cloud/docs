# دسترسی به فایل‌های Static
<hr className="mb-2" />

در NextJS می‌توانید فایل‌های استاتیک (مانند تصاویر) را در یک پوشه به نام <Important>public</Important> در مسیر اصلی پروژه قرار دهید. سپس فایل‌های موجود در مسیر public از طریق روت دامنه با شروع از <Important>/</Important> در دسترس هستند. برای مثال اگر شما یک فایل به نام example.png در پوشه‌ی public داشته باشید، می‌توانید طبق مسیر زیر به این فایل دسترسی داشته باشید:


<div className="h-2" />
<div dir='ltr'>
    <Highlight className="bash">
        {`https://yourdomain.com/example.png`}
    </Highlight>
</div>
<div className="h-2" />

در برنامه نیز، می‌توانید با استفاده از قطعه کد مثال زیر، به عکس دسترسی داشته باشید:

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`import Image from 'next/image'

export function Example() {
  return <Image src="/example.png" width="64" height="64" />
}`}
    </Highlight>
</div>
<div className="h-2" />

دایرکتوری public برای ذخیره فایل‌هایی مانند آیکون وب‌سایت، فونت‌ها، google site verification و ... مناسب است. همچنین در نظر داشته باشید که نمی‌توانید نام آن را از <Important>public</Important> به چیز دیگری، تغییر دهید و این دایرکتوری، تنها مسیری است که برای ارائه فایل‌های static، استفاده می‌شود.
