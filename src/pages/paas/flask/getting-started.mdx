# شروع به کار با پلتفرم Flask
<hr className="mb-2" />

<a href="https://flask.palletsprojects.com/en/3.0.x/" className="text-[#2196f3] ">Flask</a> یک میکرو فریم‌ورک وب برای زبان برنامه‌نویسی پایتون است که به سادگی و انعطاف‌پذیری مشهور است. Flask به دلیل طراحی مینیمال خود، تنها ابزارهای ضروری برای توسعه وب را ارائه می‌دهد و این امکان را به توسعه‌دهندگان می‌دهد که با استفاده از افزونه‌ها و کتابخانه‌های مختلف، پروژه‌های خود را به راحتی گسترش دهند.

<div className="h-2" />
