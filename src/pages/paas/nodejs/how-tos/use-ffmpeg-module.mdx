# استفاده از ماژول FFMPEG در NodeJS
<hr className="mb-2" />

<a href="https://ffmpeg.org/" className="text-[#2196f3] ">FFMPEG</a> یک نرم‌افزار متن‌باز برای پردازش چندرسانه‌ای است که قابلیت ضبط، تبدیل، پخش ویدیوی زنده و پخش استریمی (streaming) را دارد. این ابزار بسیار قدرتمند است و برای انواع مختلفی از فرمت‌های صوتی و تصویری، قابلیت‌ها و توابع متنوعی ارائه می‌دهد. 

ماژول FFMPEG به‌صورت پیش‌فرض در برنامه‌های NodeJS نصب است و همچنین متغیرهای محیطی <Important>FFMPEG_PATH</Important> و <Important>FFPROBE_PATH</Important> در این پلتفرم تنظیم شده‌اند.
بنابراین شما می‌توانید بدون انجام کار خاصی، به راحتی از این ماژول، استفاده کنید.
