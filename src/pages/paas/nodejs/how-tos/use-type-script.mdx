# استفاده از TypeScript در برنامه‌های NodeJS
<hr className="mb-2" />

تایپ‌اسکریپت یک زبان برنامه‌نویسی متن‌باز است که توسط Microsoft توسعه داده شده است و به عنوان یک زبان فرعی (superset) از JavaScript شناخته می‌شود و با استفاده از امکاناتی که ارائه می‌دهد؛ به برنامه‌نویسان کمک می‌کند تا پروژه‌های پیچیده‌تر و بزرگ‌تری را توسعه بدهند.

برای استقرار برنامه‌های NodeJS مبتنی بر TypeScript، بایستی در ابتدا در Local، در مسیر اصلی پروژه، یک دایرکتوری به نام <Important>src</Important> داشته باشید و فایل‌های با پسوند <Important>ts.</Important> در آنجا قرار داشته باشند.

با در نظر گرفتن مورد بالا، باید در مسیر اصلی پروژه، یک فایل به نام <Important>tsconfig.json</Important> ایجاد کنید و محتوای زیر را درون آن قرار دهید:

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="json">
        {`{
    "compilerOptions": {
      "target": "ES6",
      "module": "commonjs",
      "strict": true,
      "esModuleInterop": true,
      "outDir": "./dist",
      "rootDir": "./src"
    },
    "include": ["src/**/*"],
    "exclude": ["node_modules"]
  }`}
    </Highlight>
</div>
<div className="h-2" />

سپس، دستور زیر را اجرا کنید تا برنامه کامپایل شده و در پوشه <Important>dist</Important> قرار بگیرد:

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="bash">
        {`npx tsc `}
    </Highlight>
</div>
<div className="h-2" />

در ادامه، در فایل <Important>package.json</Important> اسکریپت‌های <Important>start</Important> و <Important>build</Important> را باید به شکل زیر بنویسید (فرض کنید که نام فایل اصلی پروژه، <Important>server.js</Important> است):

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="json">
        {`{
  "scripts": {
    "start": "node dist/server.js",
    "build": "tsc"
  }
}`}
    </Highlight>
</div>
<div className="h-2" />

در نهایت، می‌توانید با اجرای دستور <Important>liara deploy</Important> و یا با استفاده از Console، برنامه خود را در لیارا، مستقر کنید.

</p>
