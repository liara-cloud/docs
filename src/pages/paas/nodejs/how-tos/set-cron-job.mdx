# تنظیم Cron Job
<hr className="mb-2" />

Cron job یک وظیفه زمان‌بندی شده در سیستم‌عامل‌های Unix و Linux است که به کاربران اجازه می‌دهد تا اسکریپت‌ها یا دستورات را در فواصل زمانی منظم اجرا کنند. این ابزار به خصوص برای انجام وظایف دوره‌ای مثل پشتیبان‌گیری، ارسال ایمیل، اجرای اسکریپت‌های نگهداری سیستم، یا به‌روزرسانی اطلاعات مفید است.
<div className="h-4" />

برای تنظیم یک Cron Job در NodeJS، می‌توانید از کتابخانه‌های مختلفی استفاده کنید. یکی از محبوب‌ترین کتابخانه‌های این کار، <Important>node-cron</Important> است. 
در ادامه، به نحوه استفاده از این کتابخانه، پرداخته شده است. در ابتدا، با اجرای دستور زیر، این کتابخانه را بر روی پروژه خود، نصب کنید: 

<div className="h-2" />
<div dir="ltr">
<Highlight  className="js">
{`npm install node-cron`}
</Highlight>
</ div>
<div className="h-2" />

اکنون می‌توانید cron jobهای مختلف خود را در برنامه، تنظیم کنید؛ به عنوان مثال: 
<div className="h-2" />
<div dir="ltr">
<Highlight  className="js">
{`const cron = require('node-cron');

// create a cron to run every minute
cron.schedule('* * * * *', () => {
  console.log('Test Message');
});

// other codes ...
`}
</Highlight>
</ div>
<div className="h-2" />

<Section id="cron-format" title="فرمت زمان‌بندی Cron Job" />
در نظر داشته باشید که فرمت زمان‌بندی Cron به صورت زیر است:
<div className="h-2" />
<div dir="ltr">
<Highlight  className="scss">
{`* * * * *
│ │ │ │ │
│ │ │ │ └── روزهای هفته (۰-۷) (۰ و ۷ هر دو نشان‌دهنده یکشنبه هستند)
│ │ │ └──── ماه‌ها (۱-۱۲)
│ │ └────── روزهای ماه (۱-۳۱)
│ └──────── ساعت‌ها (۰-۲۳)
└────────── دقیقه‌ها (۰-۵۹)
`}
</Highlight>
</ div>
<div className="h-2" />

به عنوان مثال، قطعه کد زیر، برای تنظیم Cron در هر دوشنبه، ساعت 7 صبح است:

<div className="h-2" />
<div dir="ltr">
<Highlight  className="js">
{`cron.schedule('0 7 * * 1', () => {
  console.log('have a good day!');
});
`}
</Highlight>
</ div>
