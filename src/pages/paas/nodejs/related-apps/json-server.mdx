# استقرار برنامه‌های مبتنی بر JSON Server در لیارا
<hr className="mb-2" />
<a href="https://github.com/typicode/json-server" className="text-[#2196f3] ">JSON Server</a> یک ابزار ساده و کاربردی برای ساختن یک REST API موقت و سریع است که داده‌های آن بر اساس یک فایل JSON ساده می‌باشد. این ابزار برای تست و نمونه‌سازی سریع پروژه‌ها بسیار مفید است و نیازی به پیکربندی پیچیده ندارد. JSON Server به شما اجازه می‌دهد تا با داده‌های ساختگی کار کنید و درخواست‌های HTTP مانند GET , POST , PUT , PATCH و DELETE را مدیریت کنید. 
شما می‌توانید برنامه‌های مبتنی بر JSON Server خود را با <a href="../../how-tos/create-app" className="text-[#2196f3] ">ایجاد برنامه‌های NodeJS</a> در لیارا، مستقر کنید.
<div className='h-2' />

برای این کار، کافیست تا یک دایرکتوری خالی با نام دلخواه‌تان ایجاد کنید؛ وارد آن شوید و درون این دایرکتوری، فایل‌های زیر را، ایجاد کنید:

<div className="h-2" />
<ul>
    <li>فایل <Important>db.json</Important></li>
    <li>فایل <Important>json-server.json</Important></li>
    <li>فایل <Important>package.json</Important></li>
</ul>
<div className="h-2" />

اکنون کافیست تا قطعه‌کدهای مربوط به هر فایل را، درون آن‌ها، قرار دهید. قطعه کد مربوط به فایل <Important>db.json</Important>:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="json">
        {`{
    "posts": [
      { "id": 1, "title": "json-server", "author": "typicode" }
    ],
    "comments": [
      { "id": 1, "body": "some comment", "postId": 1 }
    ],
    "profile": { "name": "typicode" }
}`}
    </Highlight>
</div>
<div className="h-2" />

قطعه کد مربوط به فایل <Important>json-server.json</Important>:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="json">
        {`{
    "port": 3000,
    "host": "0.0.0.0"
} `}
    </Highlight>
</div>
<div className="h-2" />

قطعه کد مربوط به فایل <Important>package.json</Important>:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="json">
        {`{
  "devDependencies": {
    "json-server": "0.17.4"
  },
  "scripts": {
    "start": "json-server --watch db.json"
  }
}`}
    </Highlight>
</div>
<div className="h-2" />
<Alert variant="info">

در قطعه کد فوق، می‌توانید نسخه <Important>json-server</Important> مورد نظر خود را وارد کنید.
</p>
</Alert>
