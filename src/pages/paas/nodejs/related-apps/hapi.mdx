# استقرار برنامه‌های Hapi در لیارا
<hr className="mb-2" />
نرم‌افزار Hapi یک فریم‌ورک قوی و انعطاف‌پذیر برای ساخت اپلیکیشن‌های وب و API‌ها در NodeJS است. 
شما می‌توانید برنامه‌های Hapi خود را با <a href="../../how-tos/create-app" className="text-[#2196f3] ">ایجاد برنامه‌های NodeJS</a> در لیارا، مستقر کنید.

برای استقرار برنامه‌های Hapi باید اسکریپت <Important>start</Important> را در فایل <Important>package.json</Important> به شکل زیر، تعریف کنید:

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="json">
        {`{
    "scripts": {
    "test": "echo \\"Error: no test specified\\" && exit 1",
    "start": "node server.js"
  },
}`}
    </Highlight>
</div>
<div className="h-2" />

در ادامه، در فایل اصلی برنامه خود، مثلاً فایل <Important>server.js</Important>، باید حتماً فیلد <Important>host</Important> را با مقدار <Important>'0.0.0.0'</Important> مشخص کنید؛ به عنوان مثال:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`const init = async () => {
    const server = Hapi.server({
        port: 3000,
        host: '0.0.0.0',
        routes: {
            files: {
                relativeTo: Path.join(__dirname, 'public')
            }
        }
    });
    // other codes ...
};`}
    </Highlight>
</div>
<div className="h-2" />
