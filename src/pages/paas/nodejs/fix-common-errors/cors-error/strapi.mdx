# رفع خطای CORS در فریم‌ورک Strapi
<hr className="mb-2" />

برای رفع خطای CORS در Strapi، باید تنظیمات مربوط به CORS را در فایل <Important>config/middleware.js</Important> به برنامه خود اضافه کنید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`module.exports = ({ env }) => ({
  settings: {
    cors: {
      enabled: true,
      origin: '*', 
    },
  },
});`}
    </Highlight>
</div>
