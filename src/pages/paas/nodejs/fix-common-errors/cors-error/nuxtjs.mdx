# رفع خطای CORS در فریم‌ورک NuxtJS
<hr className="mb-2" />

برای رفع خطای CORS در زمان Proxy کردن درخواست‌ها در برنامه‌های NuxtJS باید در قدم اول با اجرای دستور زیر، پکیج nuxtjs/proxy را در پروژه‌ی خود نصب کنید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="bash">
        {`npm install @nuxtjs/proxy`}
    </Highlight>
</div>
<div className="h-2" />

در قدم بعد باید <Important>nuxtjs/proxy</Important> را در بخش <Important>modules</Important> فایل <Important>nuxt.config.js</Important>، اضافه کرده و پیکربندی موردنظر را اعمال کنید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`modules: [
  ...,
  '@nuxtjs/proxy'
],

axios: {
  proxy: true
},

proxy: {
  '/api/': {
      target: 'https://api.example.com/',
      pathRewrite: {'^/api/': ''},
      changeOrigin: true
    }
}`}
    </Highlight>
</div>
<div className="h-2" />
<Alert variant="info">
<p>
توجه داشته باشید که مقادیر <Important>target</Important> و <Important>pathRewrite</Important> را در پروژه‌ی خود به‌ترتیب با آدرس <Important>API</Important> و <Important>Route</Important> پروژه جایگزین کنید.
</p>
</Alert>
