# رفع خطای CORS در فریم‌ورک NestJS
<hr className="mb-2" />

برای رفع این خطا در فریم‌ورک NestJS، در ابتدا باید ماژول‌های مورد نیاز را با اجرای دستور زیر، نصب کنید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="bash">
        {`npm install @nestjs/common @nestjs/core cors`}
    </Highlight>
</div>
<div className="h-2" />

در نهایت، می‌توانید در فایل <Important>src/main.ts</Important> خود، مانند قطعه کد زیر، عمل کنید:
<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`import { NestFactory } from '@nestjs/core';
import { AppModule } from './app.module';

async function bootstrap() {
  const app = await NestFactory.create(AppModule);

  app.enableCors({
    origin: '*',
    methods: 'GET,HEAD,PUT,PATCH,POST,DELETE',
    credentials: true,
  });

  await app.listen(3000, '0.0.0.0');
}
bootstrap();`}
    </Highlight>
</div>
