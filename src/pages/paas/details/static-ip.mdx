# آی‌پی ثابت

<hr className="mb-2" />
زمانی که یک برنامه جدید در لیارا ایجاد می‌کنید، به صورت پیش‌فرض، آی‌پی برنامه‌تان متغیر خواهد بود و ممکن است به صورت دوره‌ای این IP تغییر کند.
اما برای مواردی مانند درگاه پرداخت بانکی و یا سامانه‌های پیامکی نیازمند آی‌پی ثابت هستید که این قابلیت هم وجود دارد.

<Section id="enable-static-ip" title="فعال‌کردن آی‌پی ثابت" />
لیارا این امکان را برای‌تان فراهم کرده که در برنامه خود از آی‌پی ثابت اشتراکی نیز برخوردار باشید. برای اینکه تمام درخواست‌های خروجی برنامه با یک IP ثابت صورت بگیرد. شما باید وارد قسمت تنظیمات برنامه خود شده و گزینه <b>فعال‌کردن آی‌پی ثابت</b> را روشن کنید:
<div className="h-4" />
<img src="https://files.liara.ir/liara/docs/enabled-static-ip.png" alt="enable-static-ip"/>
<div className="h-4" />
پس از انجام این کار، یک IP ثابت به برنامه شما تعلق می‌گیرد که در بخش اطلاعات کلی نیز، می‌توانید آن را مشاهده بفرمایید؛ به عنوان مثال:
<div className="h-4" />
<img src="https://files.liara.ir/liara/docs/see-static-ip-info.png" alt="see-static-ip-info"/>

<Section id='what-is-static-sharing-ip' title="آی‌پی اشتراکی چیست؟" />
آی‌پی که در اختیار شما قرار گرفته، ممکن است به چندین برنامه‌ی دیگر هم در لیارا اختصاص داده شده باشد. یعنی آی‌پی ثابتی که لیارا به شما می‌دهد اشتراکی است و مانند VPS ها اختصاصی نیست. این مورد مشکل خاصی برای‌تان ایجاد نمی‌کند و می‌توانید با خیال راحت، IP خود را به بانک یا سرویس مورد نظرتان بدهید.

<div className="h-4" />
<Alert variant="warning">
    در نظر داشته باشید که نمی‌توانید از IP ثابت برای دسترسی به سرویس خود، استفاده کنید. 
    چرا که این IP اشتراکی است و ممکن است به چندین برنامه دیگر هم اختصاص داده شده باشد. این آی‌پی فقط برای شناسایی سرور و نه ارتباط مستقیم، می‌باشد.
</Alert>
