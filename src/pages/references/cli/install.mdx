# نصب و به‌روزرسانی Liara CLI
<hr className="mb-2" />

برای نصب cli،
بایستی npm و NodeJS بر روی سیستم‌عامل‌تان نصب باشد که می‌توانید با مراجعه 
به <a href="https://nodejs.org/en/" className="text-[#2196f3] ">وب‌سایت رسمی NodeJS</a>، آخرین نسخه NodeJS TLS را متناسب با سیستم خود، دانلود و نصب کنید. پس از نصب NodeJS، ابزار npm نیز به صورت خودکار برای‌تان نصب می‌شود. 
اکنون. برای نصب Liara CLI، کافیست تا در ترمینال خود، دستور زیر را اجرا کنید: 

<div className="h-2" />
<div dir="ltr">
<Highlight className='bash'>
{`npm install -g @liara/cli`}
</Highlight>
</div>
<div className="h-2" />
<Asciinema id="install-liara-cli" />

برای ارتقاء Liara CLI نیز، بایستی همان دستور فوق را اجرا کنید. 
و درنهایت، با دستور زیر می‌توانید نسخه‌ی CLI جدید را چک کنید:

<div className="h-2" />
<div dir="ltr">
<Highlight className='bash'>
{`liara -v`}
</Highlight>
</div>
<div className="h-2" />

<Section id="command-not-found-error" title="رفع خطای عدم شناسایی Liara CLI" />
در صورتی که در اجرای دستورات Liara CLI به خطای <Important>command not found</Important> یا <Important>command is not recognized as an internal or external command</Important> برخوردید؛ کافیست تا به شکل زیر، به دستورات Liara CLI، عبارت <Important>npx</Important> را اضافه کنید: 

<div className="h-2" />
<div dir="ltr">
<Highlight className='bash'>
{`npx liara login
npx liara deploy
`}
</Highlight>
</div>
<div className="h-2" />
یا: 
<div className="h-2" />
<div dir="ltr">
<Highlight className='bash'>
{`npx @liara/cli login
npx @liara/cli deploy`}
</Highlight>
</div>
