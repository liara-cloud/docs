# مشاهده لیستی از ایمیل‌سرورهای ایجاد شده با Liara CLI
<hr className="mb-2" />

برای 
مشاهده لیستی از ایمیل‌سرورهای ایجاد شده با Liara CLI،
می‌توانید از دستور زیر، استفاده کنید: 

<div className="h-2" />
<div dir="ltr">
<Highlight className='bash'>
{`liara mail ls # or liara mail list`}
</Highlight>
</div>

    <value>] [--team-id <value>]`}
</Highlight>
</div>
