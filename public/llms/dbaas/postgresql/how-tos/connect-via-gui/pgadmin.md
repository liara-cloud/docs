# اتصال به دیتابیس PostgreSQL در PGAdmin

[PGAdmin](https://www.pgadmin.org/) یک ابزار گرافیکی برای مدیریت پایگاه‌های داده PostgreSQL است. با استفاده از pgAdmin می‌توانید پایگاه‌های داده را ایجاد و مدیریت کنید، جداول و داده‌ها را ویرایش کنید، و پرس و جوهای SQL را اجرا کنید. این ابزار قابلیت‌هایی مانند پایش عملکرد، تنظیمات پیکربندی، و ایجاد گزارش‌های مختلف را نیز فراهم می‌کند. pgAdmin به عنوان یک رابط کاربری بصری برای تسهیل کار با PostgreSQL طراحی شده است.

برای اتصال به PGAdmin کافیست تا گزینه آن را در منوی **نحوه اتصال** دیتابیس خود، فعال کنید  
برای ورود به حساب کاربری، می‌توانید از نام کاربری `root@pgadmin.org` و رمزعبور دیتابیس خود، استفاده کنید.
