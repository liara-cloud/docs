# اتصال به دیتابیس MySQL در MySQL Workbench

[MySQL Workbench](https://www.mysql.com/products/workbench/) یک ابزار گرافیکی برای مدیریت و طراحی پایگاه‌داده‌های MySQL است. این برنامه امکاناتی مانند طراحی مدل‌های ER (مدل‌های موجودیت-رابطه)، ویرایش و مدیریت جداول و داده‌ها، اجرای کوئری‌ها و تحلیل عملکرد پایگاه‌داده را فراهم می‌آورد. MySQL Workbench به توسعه‌دهندگان و مدیران پایگاه‌داده کمک می‌کند تا فرآیند طراحی و نگهداری پایگاه‌داده‌ها را به‌صورت بصری و کارآمد انجام دهند.

برای اتصال به MySQL Workbench کافیست از اطلاعات شبکه عمومی دیتابیس، که در صفحه **نحوه اتصال** موجود است؛ استفاده کنید.  
برای فعال‌سازی شبکه عمومی نیز، می‌توانید از قسمت **تنظیمات دیتابیس**، این کار را انجام دهید.
