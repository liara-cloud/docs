# اتصال به دیتابیس MSSQL در SQL Server Management Studio

[SQL Server Management Studio](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver16) یا SSMS، یک ابزار قدرتمند و جامع برای مدیریت پایگاه‌های داده SQL Server است. این ابزار به کاربران این امکان را می‌دهد تا به راحتی پایگاه‌های داده را طراحی، مدیریت و بهینه‌سازی کنند. SSMS امکانات متنوعی از جمله نوشتن و اجرای کوئری‌های SQL، طراحی جداول، مدیریت کاربران و پشتیبان‌گیری از داده‌ها را فراهم می‌آورد. همچنین، رابط کاربری گرافیکی آن کار با پایگاه‌های داده را برای توسعه‌دهندگان و مدیران پایگاه داده ساده‌تر می‌کند.

برای اتصال به SQL Server Management Studio کافیست از اطلاعات شبکه عمومی دیتابیس، که در صفحه **نحوه اتصال** موجود است؛ استفاده کنید.  
برای فعال‌سازی شبکه عمومی نیز، می‌توانید از قسمت **تنظیمات دیتابیس**، این کار را انجام دهید.
