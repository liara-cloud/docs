# اتصال به دیتابیس MongoDB در MongoDB Compass

[MongoDB Compass](https://www.mongodb.com/products/tools/compass) یک ابزار برای MongoDB است که به کاربران امکان می‌دهد به راحتی با پایگاه داده‌های MongoDB کار کنند. با استفاده از این ابزار، کاربران می‌توانند داده‌ها را مشاهده، تحلیل، و مدیریت کنند، همچنین کوئری‌های پیچیده را بدون نیاز به نوشتن کد اجرا کنند. MongoDB Compass قابلیت‌های متعددی از جمله نمایش ساختار داده‌ها، شاخص‌ها، و ابزارهای تجزیه و تحلیل عملکرد را فراهم می‌کند که به بهینه‌سازی و مدیریت بهتر پایگاه داده‌ها کمک می‌کند.

برای اتصال به MongoDB Compass کافیست از اطلاعات شبکه عمومی دیتابیس، که در صفحه **نحوه اتصال** موجود است؛ استفاده کنید.  
برای فعال‌سازی شبکه عمومی نیز، می‌توانید از قسمت **تنظیمات دیتابیس**، این کار را انجام دهید.
