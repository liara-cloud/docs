# سرویس ایمیل(سرور) لیارا

## ایمیل‌سرور چیست؟

**ایمیل سرور (Email Server)** سیستمی است که وظیفه ارسال، دریافت و مدیریت ایمیل‌ها را بر عهده دارد. این سرور شامل نرم‌افزارهای Mail Transfer Agent یا MTA برای انتقال ایمیل‌ها و Mail Delivery Agent یا MDA برای تحویل آن‌ها به صندوق پستی کاربران است.
ایمیل سرورها معمولاً از پروتکل‌های استانداردی مثل SMTP (برای ارسال ایمیل) استفاده می‌کنند.

## ایمیل تراکنشی چیست؟
ایمیل تراکنشی یا Transactional Email به ایمیل‌هایی گفته می‌شود که در پاسخ به یک عمل ارسال می‌شوند. برای مثال می‌توان ایمیل‌های تایید ثبت نام، فراموشی رمز عبور و سایر موارد مشابه را از دسته‌ی ایمیل‌های تراکنشی دانست.

## پروتکل‌های POP3/IMAP
باتوجه به اینکه سرویس ایمیل لیارا ارائه دهنده صندوق پستی (Mailbox) نیست و برای نیازمندی‌های برنامه‌نویسی طراحی شده است، امکان استفاده از پروتکل‌های POP3/IMAP وجود ندارد. 
و به جای آن، شما می‌توانید از پنل اختصاصی مدیریت ایمیل لیارا، استفاده کنید.
