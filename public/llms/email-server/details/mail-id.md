# آیدی یک ایمیل‌سرور

آیدی ایمیل سرور (mailServerId) یک شناسه منحصربه‌فرد است که لیارا برای هر سرور ایمیلی که ایجاد می‌کنید، اختصاص می‌دهد. این آیدی برای ارسال، دریافت و مدیریت ایمیل‌ها از طریق API ضروری است. هنگام ارسال درخواست به API لیارا، باید این مقدار را در URL درخواست قرار دهید تا مشخص شود که ایمیل از کدام سرور ارسال می‌شود.

## نحوه مشاهده آیدی ایمیل‌سرور
روش‌های مختلفی برای مشاهده آیدی ایمیل‌سرور مدنظرتان، وجود دارد  
که یکی از آن‌ها، استفاده از inspect است.  
در مرورگر خود، inspect را باز کنید  
و وارد سرور ایمیل مدنظرتان شوید.  
در بخش **Network**، بر روی درخواست `mails` کلیک کنید.  
وارد زیر بخش `Response` شوید و در بخش `mailServers`، `id` مربوط به سرور ایمیل مدنظرتان را مشاهده کنید.
