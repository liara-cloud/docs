# تنظیم فوروارد

فوروارد کردن (Forward) در یک ایمیل به معنای ارسال مجدد یک ایمیل دریافت شده به یک یا چند گیرنده جدید است. وقتی یک ایمیل را فوروارد می‌کنید، محتوای اصلی ایمیل به همراه هرگونه پیوست آن به گیرنده جدید ارسال می‌شود. این قابلیت به شما امکان می‌دهد تا به راحتی اطلاعات یا پیام‌های دریافت شده را با دیگران به اشتراک بگذارید.

شما می‌توانید در لیارا، برای هر نشانی ایجاد شده در ایمیل‌سرورتان، یک فوروارد، تنظیم کنید.  
کافیست تا وارد صفحه **فورواردها** شده و در بخش نشانی مدنظرتان، روی گزینه **ایجاد فوروارد** کلیک کنید.  
در ادامه، باید نشانی مقصد را وارد کرده و بر روی گزینه **افزودن نشانی** کلیک کنید تا فوروارد تنظیم شود:

![email server](https://media.liara.ir/docs/set-forward-for-email.gif)

پس از انجام کار فوق، تمامی ایمیل‌های ارسال شده به نشانی تنظیم شده، به ایمیل تعیین شده شما نیز، ارسال می‌شود.

## Liara Console

پس از بارگذاری پروژه خود در لیارا، کافیست تا در مرحله انتخاب نسخه، نسخه مورد نظر برنامه‌تان را انتخاب کنید.  
![how to choose nodejs version in liara console](https://media.liara.ir/docs/choose-nodejs-version.png)

## Liara CLI

برای تغییر نسخه پیش‌فرض، کافیست تا یک فایل به نام `liara.json` در مسیر اصلی پروژه، ایجاد کنید و قطعه کد زیر را درون آن، قرار دهید. قطعه کد زیر برای تغییر نسخه NodeJS به 18 می‌باشد:

```json
{
    "node": {
    "version": "18"
  }
}
```
