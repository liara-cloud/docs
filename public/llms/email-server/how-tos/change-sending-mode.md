# تغییر حالت ارسال ایمیل

به صورت پیش‌فرض سرور ایمیل لیارا، در حالت `DEV` ساخته می‌شود. با فعال‌بودن حالت توسعه (DEV)، ایمیل‌ها در قسمت ارسال شده‌ها باقی می‌مانند و به کاربر ارسال نمی‌شوند تا شما بتوانید در زمان توسعه و تست نرم‌افزار‌تان، به‌راحتی به هر آدرسی ایمیل ارسال کنید و نگران اسپم‌شدن سرور ایمیل‌تان نباشید.

برای تغییر حالت ارسال ایمیل‌سرور به `Production` کافیست تا وارد قسمت **تنظیمات** شوید و گزینه **حالت توسعه‌دهنده** را، خاموش کنید:

![email server](https://media.liara.ir/docs/turn-off-dev-mode-email-server.png)

با انجام کار فوق، سرور ایمیل شما به `LIVE` تغییر یافته و ایمیل‌های شما به ایمیل مقصد ارسال می‌شوند.

- نسخه 18
- نسخه 20 (پیش‌فرض)

## Liara Console

پس از بارگذاری پروژه خود در لیارا، کافیست تا در مرحله انتخاب نسخه، نسخه مورد نظر برنامه‌تان را انتخاب کنید.  
![how to choose nodejs version in liara console](https://media.liara.ir/docs/choose-nodejs-version.png)

## Liara CLI

برای تغییر نسخه پیش‌فرض، کافیست تا یک فایل به نام `liara.json` در مسیر اصلی پروژه، ایجاد کنید و قطعه کد زیر را درون آن، قرار دهید. قطعه کد زیر برای تغییر نسخه NodeJS به 18 می‌باشد:

```json
{
    "node": {
    "version": "18"
  }
}
```
