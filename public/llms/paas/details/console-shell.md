# خط فرمان کنسول

خط فرمان ابزاری است که به شما امکان می‌دهد تا به‌طور مستقیم با سرور خود ارتباط برقرار کنید و دستورات مختلف را برای مدیریت و تنظیم سرور و برنامه‌های مختلف وارد کنید.  
در برخی از برنامه‌ها، پس از استقرار پروژه، برای اجرای موفق، بایستی یک‌سری دستورات در خط فرمان، اجرا شوند. به عنوان مثال، ممکن است بخواهید دیتابیس را با داده‌های اولیه پر کنید و یا ساختار جداول را ایجاد کنید (migrationها را اعمال کنید). خط فرمان بهترین جایی است که می‌توانید دستوراتی که معمولا یک بار لازم است اجرا شوند را اجرا کنید.

## Liara Console

برای اتصال به خط فرمان برنامه خود، کافیست تا وارد بخش **خط فرمان** شوید و سپس بر روی گزینه اتصال با مرورگر کلیک کنید تا خط فرمان برای‌تان بالا بیاید.

[Video link](https://media.liara.ir/docs/connect-to-shell-via-console.mp4)

> اگر که از [پلتفرم داکر لیارا](https://docs.liara.ir/paas/docker/getting-started) استفاده می‌کنید و `bash` در `image` شما نصب نیست؛ می‌توانید از `shell` پیش‌فرض `image` استفاده کنید.

این ترمینال،`bin/bash/`را اجرا می‌کند. بنابراین می‌توانید هر دستوری که در `bash` قابل اجرا است را وارد نمایید.

## Liara CLI

برای اجرای دستور‌های مورد نیاز در فرایند [CI/CD](https://docs.liara.ir/paas/cicd/about) و یا دسترسی مستقیم به خط فرمان برنامه در سیستم‌عامل خود، می‌توانید از دستور زیر استفاده کنید:

```bash
liara shell
```

پس از اجرای دستور فوق و انتخاب برنامه‌ی مورد نظر به‌صورت مستقیم به خط فرمان برنامه با استفاده از `bin/bash/` متصل خواهید شد اما درصورتی که تصمیم داشته باشید دستور به‌خصوصی را اجرا کنید یا در برنامه‌های [Docker](https://docs.liara.ir/paas/docker/getting-started) به `bin/sh/` متصل شوید کافیست command فوق را به شکل زیر تغییر دهید:

```bash
liara shell --command=/bin/sh
```

## مشکل قطع مداوم اتصال به خط فرمان

برای این‌که بتوانید به‌خط فرمان وصل شوید، برنامه‌ی شما باید روشن و در حال اجرا باشد. برای اطمینان از اجرای موفق برنامه، کافیست تا [گزارشات](../observations/about) و [رویدادها](../events) مربوط به برنامه را بررسی کنید. چنانچه برنامه‌ی شما مداوم در حال crash کردن و ری‌استارت‌شدن باشد، امکان اتصال به‌خط فرمان را نخواهید داشت و ارتباط شما قطع خواهد شد.

## نصب پکیج‌های دلخواه بر روی سرور با استفاده از apt-get

در نظر داشته باشید که نمی‌توانید با استفاده از خط فرمان و دستور `apt-get` پکیج مدنظرتان را بر روی سرور نصب کنید. اما می‌توانید با استفاده از Hookها (در صورت پشتیبانی پلتفرم) و یا [پلتفرم Docker](https://docs.liara.ir/paas/docker/getting-started)، هرگونه شخصی‌سازی که نیاز دارید را، انجام دهید و پکیج‌های مدنظرتان را، نصب کنید.
