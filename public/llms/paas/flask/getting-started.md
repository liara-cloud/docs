# شروع به کار با پلتفرم Flask

[Flask](https://flask.palletsprojects.com/en/3.0.x/) یک میکرو فریم‌ورک وب برای زبان برنامه‌نویسی پایتون است که به سادگی و انعطاف‌پذیری مشهور است. Flask به دلیل طراحی مینیمال خود، تنها ابزارهای ضروری برای توسعه وب را ارائه می‌دهد و این امکان را به توسعه‌دهندگان می‌دهد که با استفاده از افزونه‌ها و کتابخانه‌های مختلف، پروژه‌های خود را به راحتی گسترش دهند.

فریم‌ورک Flask بر پایه WSGI یا Web Server Gateway Interface ساخته شده و برای پروژه‌های کوچک تا متوسط مناسب است. Flask همچنین از سرور توسعه داخلی، مسیریابی URL و قالب‌های Jinja2 پشتیبانی می‌کند.
شما می‌توانید در سریع‌ترین زمان ممکن و به سادگی چند کلیک یا اجرای چند دستور ساده، برنامه خود را در لیارا مستقر کنید. در ادامه بخوانید:
