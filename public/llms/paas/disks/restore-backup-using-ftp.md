# بازیابی فایل پشتیبان با دسترسی FTP در دیسک

برای بازیابی فایل پشتیبان، کافیست تا یک  
دسترسی FTP برای دیسک مدنظرتان ایجاد کنید. با استفاده از دسترسی ایجاد شده، به دیسک متصل شوید و  
محتوای درون فایل پشتیبان (و نه خود فایل فشرده) را، درون دیسک، بریزید.  
در مثال زیر، محتوای درون فایل فشرده با استفاده از دسترسی FTP و نرم‌افزار FileZilla به دیسک ریخته می‌شود:

![restore backups using ftp in disks](https://media.liara.ir/docs/restore-backup-in-disks.gif)

> برای استفاده از دسترسی FTP، دست کم در یک استقرار باید از دیسک استفاده کرده باشید و وضعیت آن از آماده به کار به در حال استفاده تغییر کرده باشد.

> همچنین بخوانید: [اتصال به دیسک‌ها با استفاده از دسترسی FTP](../ftp-access)
