# استفاده از قابلیت cgo در برنامه‌های Go

قابلیت cgo به شما این امکان را می‌دهد که در برنامه‌های go، از کد مبتنی بر زبان C استفاده کنید. این قابلیت به شما کمک می‌کند که بتوانید  
کتابخانه‌های نوشته شده در زبان C را در Go به کار ببرید و یا عملکرد برنامه خود را بهبود دهید.  

این قابلیت به صورت خودکار در برنامه‌های Go در لیارا غیر فعال است. اما برای فعال‌سازی آن می‌توانید در استقرار، به روش زیر، عمل کنید:  

## Liara Console

پس از بارگذاری پروژه خود در لیارا، کافیست تا در مرحله پیکربندی پروژه، این قابلیت را فعال کنید.  
{/* ![how to choose version in liara console](https://media.liara.ir/docs/change-go-version-in-go-using-console.png) */}

## Liara CLI

برای فعال‌سازی قابلیت cgo در برنامه‌تان، کافیست تا یک فایل به نام `liara.json` در مسیر اصلی پروژه، ایجاد کنید و قطعه کد زیر را درون آن، قرار دهید:

```json
{
  "go" : {
    "cgoEnabled": "true"
   }
}
```

## Github

برای فعال‌سازی قابلیت cgo در برنامه‌تان، کافیست تا یک فایل به نام `liara.json` در مسیر اصلی پروژه، ایجاد کنید و قطعه کد زیر را درون آن، قرار دهید:

```json
{
  "go" : {
    "cgoEnabled": "true"
   }
}
```

در صورت نیاز به کتابخانه یا هر ماژول خاص دیگر که به صورت built-in در نسخه مدنظرتان در go وجود ندارد؛ بایستی  
طبق [مستندات Hookها](https://docs.liara.ir/paas/go/how-tos/use-hooks/)، دستورات مورد نیاز برای نصب ماژول‌ها را در فایل `liara_post_build.sh`، بنویسید.
