# استفاده از متغیرهای محیطی 

برای استفاده از متغیرهای محیطی در برنامه خود، در ابتدا باید طبق [مستندات تنظیم متغیرهای محیطی](https://docs.liara.ir/paas/details/envs)، متغیرهای محیطی را به برنامه خود، اضافه کنید.
در ادامه، شما می‌توانید با استفاده از متد `()env` به متغیرهای محیطی خود در برنامه Laravel، دسترسی داشته باشید؛ به عنوان مثال:

```js
$appName = env('LIARA_URL', 'my-web-app');
```

در مثال فوق، `LIARA_URL` یک متغیر محیطی است که از قبل، به برنامه فرضی در لیارا، اضافه شده است و `my-web-app` مقدار پیش‌فرضی است که در صورت تعریف نشدن متغیر محیطی `LIARA_URL`، مورد استفاده قرار می‌گیرد.

## استفاده از متغیرهای محیطی در فایل‌های کانفیگ
برای اضافه کردن متغیرهای محیطی در فایل‌های کانفیگ لاراول نیز، می‌توانید از متد `()env` استفاده کنید:

```php
'name' => env('LIARA_URL', 'my-web-app');
```

پس از اضافه کردن متغیرهای محیطی به برنامه، لیارا دستورات زیر را برای‌تان به صورت خودکار، اجرا می‌کند و نیازی نیست که شما، کار خاصی را انجام دهید:

```php
php artisan config:clear
php artisan config:cache
```
