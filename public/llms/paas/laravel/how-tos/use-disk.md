# استفاده از دیسک در برنامه‌های Laravel

[Video link](https://media.liara.ir/laravel/laravel-disks.mp4)

برای استفاده از [دیسک‌ها](https://docs.liara.ir/paas/disks/about) در برنامه‌های Laravel در ابتدا باید طبق مستندات [ساخت دیسک](https://docs.liara.ir/paas/disks/create)، دیسک‌های مدنظر خود را بسازید؛ در ادامه، می‌توانید طبق مستندات [تعریف مسیر برای دیسک](https://docs.liara.ir/paas/disks/route)، دیسک‌های خود را به دایرکتوری‌های مدنظرتان، متصل کنید.

بهتر است که مسیر دایرکتوری مدنظر خود را در برنامه‌های Laravel به صورت **کامل و مطلق** مشخص کنید؛ به عنوان مثال، برای اتصال یک دیسک به دایرکتوری `public/uploads` کافیست تا مسیر زیر را به دیسک، متصل کنید:

```bash
/var/www/html/public/uploads
```
