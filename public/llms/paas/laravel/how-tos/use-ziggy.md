# استفاده از Ziggy در برنامه‌های Laravel

Ziggy یک پکیج لاراول است که به شما امکان می‌دهد تا به راحتی به مسیر (route)های تعریف‌شده در لاراول از طریق جاوااسکریپت دسترسی داشته باشید. این ابزار برای توسعه‌دهندگانی که از فریم‌ورک‌های جاوااسکریپت سمت کاربر مانند [Vue](https://docs.liara.ir/paas/vue/getting-started) یا [React](https://docs.liara.ir/paas/react/getting-started) استفاده می‌کنند، بسیار مفید است.

برای استفاده از Ziggy در لیارا تنها کافیست فایل مورد نظرتان را از مسیر `vendor/tightenco/ziggy/dist` در پوشه‌ی `resources/js` کپی کنید. برای مثال، درصورتی که فایل `vue.m` را کپی کرده باشید؛ می‌توانید به‌شکل زیر آن را در برنامه‌ی خود import کنید:

```config
import { ZiggyVue } from './vue.m';
```
