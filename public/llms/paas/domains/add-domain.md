# اضافه کردن دامنه خریداری شده به برنامه

برای اتصال دامنه به برنامه مورد نظرتان در لیارا باید وارد برنامه شوید و از منوی سمت راست، روی گزینه‌ی **دامنه‌ها** کلیک کنید. سپس روی گزینه‌ی **افزودن دامنه** کلیک کرده و نام دامنه‌تان (مثلاً: mydomain.com) را وارد کنید. درنهایت با کلیک روی گزینه‌ی **ایجاد دامنه**، دامنه شما به برنامه اضافه می‌شود و همچنین به صفحه‌ی مدیریت دامنه هدایت می‌شوید.

[Video link](https://media.liara.ir/domain/add-domain.mp4)

با انجام کار فوق، سه رکورد DNS به شما داده می‌شود: 

- رکورد اول از نوع `ALIAS / CNAME`: این رکورد برای هدایت دامنه اصلی به سرور لیارا جهت سرویس‌دهی وب‌سایت یا اپلیکیشن استفاده می‌شود
- رکورد دوم از نوع `TXT`: این رکورد برای احراز هویت دامنه توسط لیارا، جهت اتصال اولیه دامنه و ثبت مالکیت آن کاربرد دارد
- رکورد سوم از نوع `CNAME (DNS-01 Challenge)`: این رکورد جهت اعتبارسنجی مالکیت دامنه در فرآیند صدور یا تمدید گواهی SSL مورد استفاده قرار می‌گیرد

در این مرحله باید رکوردهای DNS درخواست شده را با استفاده از یک سرویس DNS مانند [Cloudflare](https://www.cloudflare.com/) بر روی دامنه‌تان تنظیم کنید.  
درنهایت پس از ثبت رکوردهای DNS می‌توانید بر روی گزینه‌ی  **بررسی وضعیت رکوردها** کلیک کنید و با تغییر تیک‌های قرمز رنگ به سبز، یعنی اتصال دامنه‌ی شما به لیارا با موفقیت صورت گرفته است:

![activated-domain](https://media.liara.ir/docs/activated-domain.png)

در حین ثبت دامنه، نکات زیر را حتماً در نظر بگیرید:
- در صورتی که قصد دارید از CDN استفاده کنید، به [این مستندات](https://docs.liara.ir/paas/domains/use-cdn) مراجعه کنید
- درصورت استفاده از CDN می‌توانید تیک قرمز رنگ و خطای تنظیم نشدن رکوردها را نادیده بگیرید

> شما می‌توانید برای مدیریت دامنه خود، از [سرویس DNS لیارا](https://docs.liara.ir/dns-management-system/about) استفاده کنید تا رکوردها به صورت خودکار به آن اضافه شوند.
