# استقرار برنامه‌های Yii در لیارا

[Yii](https://www.yiiframework.com/) یک فریم‌ورک قدرتمند و سریع برای توسعه وب‌اپلیکیشن‌های PHP است که با تمرکز بر سادگی و عملکرد، ابزارها و ویژگی‌های متعددی برای توسعه سریع و کارآمد اپلیکیشن‌ها ارائه می‌دهد. Yii به توسعه‌دهندگان کمک می‌کند تا کدهای بهینه‌تر و ساده‌تری بنویسند.  
شما می‌توانید پروژه‌های Yii خود را با [ایجاد برنامه‌های PHP](../../how-tos/create-app) بر روی لیارا مستقر کنید. لیارا، از این فریم‌ورک به صورت کامل پشتیبانی می‌کند و فقط بایستی چندین گام را طی کنید  
تا برنامه‌تان بدون کمترین مشکلی، در لیارا، مستقر شود:

از آنجایی که ساختار فریم‌ورک Yii به‌گونه‌ای است که فایل‌های وب برنامه در پوشه‌ی web قرار داده می‌شوند؛ در ابتدا، طبق مستندات [تنظیم متغیرهای محیطی](https://docs.liara.ir/paas/details/envs) بایستی متغیر با مقدار زیر را به برنامه خود، اضافه کنید:

```bash
APACHE_DOCUMENT_ROOT=web/
```

در ادامه، برای مواجه نشدن با خطای `The directory is not writable`، باید طبق [مستندات ایجاد دیسک](https://docs.liara.ir/paas/disks/create)، یک دیسک با نام `data` و اندازه دلخواه ایجاد کرده  
و سپس، در مسیر اصلی پروژه، یک فایل به نام `liara.json` ایجاد کنید و قطعه کد زیر را در آن، قرار دهید تا دیسک به برنامه، متصل شود:

```json
{
  "disks": [
    {
      "name": "data",
      "mountTo": "web/assets"
    }
  ]
}
```

در نهایت کافیست برنامه‌تان را با دستور `liara deploy` در لیارا مستقر کنید.

> یک پروژه Yii آماده به استقرار در [اینجا](https://github.com/liara-cloud/yii-getting-started) وجود دارد که می‌توانید از آن، استفاده کنید.
