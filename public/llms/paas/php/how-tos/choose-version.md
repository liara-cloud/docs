# انتخاب نسخه PHP 

در حال حاضر در پلتفرم لیارا، نسخه‌های زیر از  PHP ارائه می‌شوند: 

- نسخه 7.2 (بدون به‌روزرسانی)
- نسخه 7.3 (بدون به‌روزرسانی)
- نسخه 7.4
- نسخه 8.0 (پیش‌فرض)
- نسخه 8.1
- نسخه 8.2
- نسخه 8.3

> قابلیت‌های جدید برای نسخه‌های بدون به‌روزرسانی، ارائه نمی‌شوند و برای رفع این مشکل، باید ورژن PHP برنامه خود را، ارتقاء دهید.

> برای تغییر نسخه PHP در برنامه‌های لاراول، می‌توانید از [این مستندات](https://docs.liara.ir/paas/laravel/how-tos/choose-version) استفاده کنید.

## Liara Console

پس از بارگذاری پروژه خود در لیارا، کافیست تا در مرحله انتخاب نسخه، نسخه مورد نظر برنامه‌تان را انتخاب کنید.
![how to choose version in liara console](https://media.liara.ir/docs/choose-php-version-of-php-app-via-cosnole.png)

## Liara CLI

برای تغییر نسخه پیش‌فرض، کافیست تا یک فایل به نام `liara.json` در مسیر اصلی پروژه، ایجاد کنید و قطعه کد زیر را درون آن، قرار دهید. قطعه کد زیر برای تغییر نسخه PHP به 8.2 می‌باشد:

```json
{
    "php": {
      "version": "8.2"
  }
}
```
