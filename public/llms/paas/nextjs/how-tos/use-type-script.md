# استفاده از TypeScript در برنامه‌های NextJS

برای استقرار برنامه‌های NextJS مبتنی بر TypeScript نیاز به انجام کار خاصی نیست و پلتفرم NextJS لیارا به صورت کامل  
از آن، پشتیبانی می‌کند. تنها نکته قابل توجه این است که فایل `package.json` برنامه‌تان باید در `devDependencies`شامل ماژول TypeScript باشد. به عنوان مثال:

```json
"devDependencies": {
"typescript": "^5",
"@types/node": "^20",
"@types/react": "^18",
"@types/react-dom": "^18",
"eslint": "^8",
"eslint-config-next": "14.2.3"
}
```
