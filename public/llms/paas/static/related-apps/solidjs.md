# استقرار برنامه‌های SolidJS در لیارا

[SolidJS](https://docs.solidjs.com/)
یک فریمورک جاوااسکریپت برای ساخت رابط‌های کاربری (UI) است که به طور خاص بر روی سرعت و کارایی متمرکز است. این فریمورک از الگوی واکنش‌گرایی (reactivity) استفاده می‌کند تا با استفاده از تغییرات دقیق و کارآمد در DOM، به سرعت بسیار بالایی دست یابد. برخلاف برخی فریمورک‌های دیگر مثل React، SolidJS از ابتدا طراحی شده تا تنها کامپوننت‌های تغییر یافته را دوباره رندر کند، که این ویژگی باعث می‌شود عملکرد بهینه‌تری داشته باشد.

شما می‌توانید برنامه‌های SolidJS خود را با [ایجاد برنامه‌های Static](../../how-tos/create-app) در لیارا، مستقر کنید.
برای استقرار برنامه‌های SolidJS 
کافیست تا در مسیر اصلی پروژه در لوکال، دستور زیر را اجرا کنید:

```bash
npm run build
```

بعد از انجام این کار، یک دایرکتوری به نام `dist` برای شما ایجاد می‌شود.  

## استقرار برنامه

### Liara Console

در نهایت کافیست تا دایرکتوری `dist` را درون یک فایل zip قرار دهید و برنامه خود را با کنسول در لیارا آپلود کنید تا برنامه با موفقیت در لیارا مستقر شود.

### Liara CLI

در نهایت کافیست تا وارد دایرکتوری `dist` شوید و دستور زیر را اجرا کنید تا برنامه شما در لیارا مستقر شود:

```bash
liara deploy --platform=static
```

> یک پروژه SolidJS آماده به استقرار در [اینجا](https://github.com/liara-cloud/solidjs-getting-started) وجود دارد که می‌توانید از آن، استفاده کنید.
