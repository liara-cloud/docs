# استقرار برنامه‌های Gridsome در لیارا

[Gridsome](https://gridsome.org/) یک فریم‌ورک متن‌باز برای ساخت وب‌سایت‌های ایستا و برنامه‌های وب با استفاده از Vue است. این فریم‌ورک با بهره‌گیری از GraphQL به توسعه‌دهندگان امکان می‌دهد داده‌ها را از منابع مختلف دریافت و مدیریت کنند. Gridsome به بهینه‌سازی عملکرد و SEO کمک می‌کند و تجربه کاربری سریعی را ارائه می‌دهد. 

شما می‌توانید برنامه‌های Gridsome خود را با [ایجاد برنامه‌های Static](../../how-tos/create-app) در لیارا، مستقر کنید.
برای این‌کار، کافیست تا در مسیر اصلی پروژه در Local، دستور زیر را اجرا کنید:

```json
npm run build
```

با اجرای دستور فوق، فرایند build گرفتن از برنامه‌ی شما شروع خواهد شد و درنهایت تمام فایل‌های استاتیک اعم از فایل‌های `HTML/CSS/JS` در پوشه‌ی `dist` قرار داده می‌شوند.
اکنون، کافیست تا وارد پوشه dist شوید و قطعه کد زیر را در آن، اجرا کنید:

```js
liara deploy --platform=static
```

با اجرای دستور فوق، برنامه‌تان در لیارا، مستقر خواهد شد.
