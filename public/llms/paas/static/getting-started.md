# شروع به کار با پلتفرم Static

یک پروژه استاتیک می‌تواند شامل یک‌سری فایل HTML , CSS یا JavaScript باشد که بدون نیاز به پردازش سرور، به مرورگر کاربران، ارسال می‌شود.  
شما می‌توانید در سریع‌ترین زمان ممکن و به سادگی چند کلیک یا اجرای چند دستور ساده، برنامه خود را در لیارا مستقر کنید. در ادامه بخوانید:

- #### [استقرار سریع یک برنامه Static در لیارا](./quick-start)
- #### [استقرار قدم به قدم برنامه‌های Static در لیارا](./how-tos/create-app)

لیارا به‌طور کامل از تمامی فریم‌ورک‌های static-site-generator و ابزارهای مرتبط با آن‌ها، پشتیبانی می‌کند و شما می‌توانید با خیال راحت از قدرت و انعطاف‌پذیری این ابزارها برای توسعه پروژه‌های خود استفاده کنید؛  
در ادامه، مستندات مربوط به ابزارها و فریم‌ورک‌های مختلف برای شما قرار گرفته است:
