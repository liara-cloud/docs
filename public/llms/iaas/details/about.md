# ویژگی‌های یک سرور مجازی ابری در لیارا

لیارا همواره سعی داشته است که با  
ارائه یک رابط کاربری آسان و همه‌منظوره، نیازهای کاربران خود را تا حد زیادی در خود [صفحه کنسول](https://console.liara.ir)، شناسایی و بر طرف کند. اما گاهاً ممکن است که در برخی از موارد، دچار ابهام شوید و به دنبال پاسخی  
برای سوالات خود، باشید.  
از این رو، در ادامه، تمامی جزئیات مربوط به سرور مجازی ابری برای شما آماده شده‌است تا با مطالعه آن‌ها، دانش شما، بیش از پیش، ارتقا یابد:

[Video link](https://media.liara.ir/vps/cloud-server-introduction.mp4)

- #### پلن‌های سرور مجازی ابری  
  آشنایی با پلن‌های سخت‌افزاری یک سرور مجازی ابری و مشاهده تعداد هسته‌های CPU، میزان RAM و مقدار Disk  
  [بیشتر بدانید](./hardware-plans)  

- #### آی‌پی سرورابری  
  آی‌پی ورژن چهار که با استفاده از آن، می‌توانید به صورت مستقیم، به سرور مجازی ابری دسترسی داشته باشید.  
  [بیشتر بدانید](./ip)  

- #### محدودیت‌های سرور مجازی ابری  
  آشنایی با محدودیت‌های سرور مجازی ابری  
  [بیشتر بدانید](https://docs.liara.ir/iaas/details/limitations)  

- #### رویدادها  
  مشاهده وضعیت نهایی سرور مجازی ابری برای بررسی تغییرات بر اساس زمان آخرین تغییرات  
  [بیشتر بدانید](./events)  

- #### تغییر پلن سرور مجازی ابری  
  تغییر پلن سرور مجازی ابری برای افزایش یا کاهش منابع سخت‌افزاری  
  [بیشتر بدانید](./change-plan)  

- #### حذف یک سرور مجازی ابری  
  راهنمایی برای حذف یک سرور مجازی ابری از حساب کاربری و پایان روزهای طلایی استفاده از آن   
  [بیشتر بدانید](./delete-server)  

- #### ابزار OpenSSH  
  نحوه نصب ابزار OpenSSH بر روی سیستم‌عامل‌های مختلف ویندوز، لینوکس و مک برای کار با SSH  
  [بیشتر بدانید](./openssh-package)  

- #### آیدی یک سرور مجازی  
  نحوه مشاهده آیدی سرور مجازی ابری برای کار با آن از طریق API لیارا  
  [بیشتر بدانید](./vm-id)  

- #### کلید SSH  
  نحوه استفاده از کلید SSH برای اتصال خیلی راحت‌تر به سرور مجازی ابری  
  [بیشتر بدانید](./ssh-key)  

- #### سیگنال‌های سرور مجازی ابری  
  آشنایی با سیگنال‌های مختلف سرور مجازی ابری و نحوه استفاده از آن‌ها  
  [بیشتر بدانید](./signals)  

- #### بازنشانی رمز root  
  آشنایی با سیگنال‌های مختلف سرور مجازی ابری و نحوه استفاده از آن‌ها  
  [بیشتر بدانید](./reset-root-password)  

- #### ‌گزارشات شبکه‌ای  
  مشاهده گزارشات شبکه‌ای سرور مجازی ابری برای بررسی میزان مصرف ترافیک دریافتی و ارسالی  
  [بیشتر بدانید](https://docs.liara.ir/iaas/details/network-observation)
