# API سرور مجازی ابری لیارا

API سرور مجازی ابری امکان مدیریت خودکار ماشین‌های مجازی را از طریق درخواست‌های HTTP فراهم می‌کند. با استفاده از این API، کاربران می‌توانند سرور جدید ایجاد کرده، وضعیت آن را بررسی کنند و سیگنال‌های قطع برق، روشن شدن، خاموش شدن و ریبوت را به آن، ارسال کنند.  
در ادامه، بخوانید:

- #### [ایجاد سرور مجازی ابری با API](https://docs.liara.ir/iaas/api/create-vm)
- #### [مشاهده رویدادهای سرور مجازی ابری با API](https://docs.liara.ir/iaas/api/see-vm-events)
- #### [ارسال سیگنال به سرور مجازی ابری با API](https://docs.liara.ir/iaas/api/send-vm-signals)
