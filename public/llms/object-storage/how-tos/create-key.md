# ایجاد کلید برای باکت

در سیستم‌های ذخیره‌سازی ابری، Access Key و Secret Key دو عنصر مهم برای احراز هویت و دسترسی به منابع هستند:
Access Key شناسه‌ای یکتاست که کاربر یا برنامه را شناسایی می‌کند و Secret Key کلید محرمانه‌ای است که همراه با Access Key استفاده می‌شود تا درخواست‌ها را اعتبارسنجی کند و اطمینان حاصل شود که درخواست از یک منبع معتبر است.

برای اتصال به ذخیره‌سازی ابری با استفاده از AWS SDK، به کلیدهای Access Key و Secret Key نیاز خواهید داشت. حال برای ایجاد کلید باید وارد بخش **کلیدها** شوید و روی گزینه‌ی **ایجاد کلید** کلیک کنید. در قدم بعد، باید توضیحاتی برای کلید قرار دهید و همچنین دسترسی کلید به باکت‌های مورد نظرتان را مشخص کنید. درنهایت می‌توانید روی گزینه‌ی **تایید**، کلیک کنید تا کلیدهای Access Key و Secret Key ساخته شوند:

[Video link](https://media.liara.ir/object-storage/generate-key.mp4)

> Secret Key تنها یک‌بار به شما نمایش داده می‌شود بنابراین آن را در جایی مطمئن ذخیره کنید.
