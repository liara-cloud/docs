# ساخت کلید جدید برای باکت

درصورتی که به‌هر دلیلی Secret Key را از دست بدهید یا نیاز به یک Secret Key جدید داشته باشید؛
می‌توانید آن را مجدداً generate کنید. برای این کار،
باید روی سه نقطه کنار کلید مدنظرتان، کلیک کرده و پس از کلیک روی گزینه‌ی **کلید جدید**، درنهایت روی گزینه‌ی **ساخت کلید جدید** کلیک کنید:

[Video link](https://media.liara.ir/object-storage/revoke-key.mp4)

> با ساخت کلید جدید، Access Key تغییری نمی‌کند و Secret Key قدیمی، بلا استفاده خواهد شد.
