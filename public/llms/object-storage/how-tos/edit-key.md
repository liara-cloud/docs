# ویرایش کلید

برای ویرایش کلید باید روی سه نقطه کنار کلید مدنظرتان، کلیک کرده و پس از کلیک روی گزینه‌ی **ویرایش**،  
می‌توانید توضیحات مربوط به کلید را تغییر دهید و یا باکت‌های مدنظرتان را به آن اضافه، یا از آن، حذف کنید.  

[Video link](https://media.liara.ir/object-storage/edit-key.mp4)

> با حذف کلید، امکان اتصال از طریق آن‌ها به باکت، دیگر وجود نخواهد داشت
