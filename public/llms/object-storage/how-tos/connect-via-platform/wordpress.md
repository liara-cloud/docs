# اتصال به فضای ذخیره‌سازی ابری در برنامه‌های WordPress

[Video link](https://media.liara.ir/wordpress/wordpress-object-storage.mp4)

برای استفاده از Object Storage در برنامه‌های WordPress، می‌توانید از  
پلاگین‌هایی مانند [`Media Cloud`](https://wordpress.org/plugins/ilab-media-tools/) استفاده کنید.  
برای نصب، فعال‌سازی و راه‌اندازی این پلاگین، می‌توانید مانند آموزش زیر، عمل کنید:
