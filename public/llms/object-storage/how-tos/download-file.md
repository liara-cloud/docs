# دانلود فایل از باکت

برای دانلود فایل  
دو روش وجود دارد:  

۱. دانلود فایل با استفاده از S3 SDK  
برای یادگیری کار با S3 SDK به [مستندات مربوطه](https://docs.liara.ir/object-storage/how-tos/connect-via-platform/about/) مراجعه بفرمایید.

۲. دانلود فایل با دامنه شخصی  
برای دانلود فایل با دامنه شخصی‌تان، کافیست تا طبق [مستندات اتصال دامنه به باکت](https://docs.liara.ir/object-storage/add-domain/)، دامنه خود را به باکت متصل کنید. سپس می‌توانید فایل خود را با لینک دامنه‌تان، دانلود کنید.

به عنوان مثال، فرض کنید که دامنه‌تان `example.com` است.  
با فرض اینکه فایل مدنظرتان در باکت در مسیر `path/to/myfile.txt` قرار دارد و دامنه‌تان به باکت متصل و SSL آن نیز فعال است، می‌توانید با مراجعه به لینک زیر، فایل را مشاهده و دانلود کنید:

```bash
https://example.com/path/to/myfile.txt
```
