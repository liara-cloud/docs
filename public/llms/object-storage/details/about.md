# ویژگی‌های Object Storage در لیارا

لیارا همواره سعی داشته است که با  
ارائه یک رابط کاربری آسان و همه‌منظوره، نیازهای کاربران خود را تا حد زیادی در خود [صفحه کنسول](https://console.liara.ir)، شناسایی و بر طرف کند. اما گاهاً ممکن است که در برخی از موارد، دچار ابهام شوید و به دنبال پاسخی  
برای سوالات خود، باشید.  
از این رو، در ادامه، تمامی جزئیات مربوط به Object Storage برای شما آماده شده‌است تا با مطالعه آن‌ها، دانش شما، بیش از پیش، ارتقا یابد:

- #### پلن‌های Object Storage  
  آشنایی با پلن‌های یک Object Storage  
  [بیشتر بدانید](./plans)

- #### گزارشات  
  مانیتور و نظارت دقیق بر فضای ذخیره‌سازی ابری  
  [بیشتر بدانید](./observations)

- #### تغییر پلن در Object Storage  
  امکان تغییر پلن‌های یک Object Storage در هر زمانی، با توجه به نیازهای یک پروژه  
  [بیشتر بدانید](./change-plan)

- #### حذف باکت  
  امکان حذف باکت در لیارا، پس از اتمام روزهای طلایی استفاده از آن  
  [بیشتر بدانید](./delete-object-storage)
