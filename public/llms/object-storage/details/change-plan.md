# تغییر پلن Object Storage در لیارا

برای تغییر پلن Object Storage خود در لیارا، کافیست تا وارد قسمت **تنظیمات**  باکت خود، شوید. در بخش **تغییر اندازه**، پلن مد نظر خود را انتخاب کنید و در نهایت بر روی گزینه **تایید**، کلیک کنید:

[Video link](https://media.liara.ir/object-storage/change-size.mp4)

> با ارتقای پلن، امکان بازگشت به پلن قبلی وجود نخواهد داشت.
