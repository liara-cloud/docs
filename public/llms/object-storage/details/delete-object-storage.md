# حذف باکت

برای حذف Object Storage، در ابتدا بایستی طبق [مستندات حذف کلید](../how-tos/delete-key)، کلیدهای متصل به باکت مدنظرتان را حذف کنید.  
یا اینکه می‌توانید بر روی گزینه **ویرایش** یک کلید کلیک کرده و باکت مورد نظرتان را از آن کلید، جدا کنید.

سپس، کافیست تا در منوی **ذخیره‌سازی ابری** بر روی سه‌نقطه باکت مدنظرتان کلیک کنید،  
سپس؛ بر روی گزینه **حذف باکت** کلیک کرده  
در نهایت، کافیست تا شناسه باکت خود را در فیلد خواسته‌شده، وارد کنید و بر روی گزینه **حذف باکت** کلیک کنید تا Object Storage‌تان حذف شود:

[Video link](https://media.liara.ir/object-storage/delete-bucket.mp4)

> با حذف فضای ذخیره‌سازی ابری، تمامی اطلاعات درون Object Storage‌تان  
> **برای همیشه** پاک خواهد شد و امکان بازیابی آن‌ها، در لیارا، وجود نخواهد داشت.
