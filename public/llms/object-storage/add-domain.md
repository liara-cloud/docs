# اضافه کردن دامنه خریداری شده به باکت

برای اتصال دامنه به باکت مورد نظرتان در لیارا باید وارد باکت شوید و سپس روی گزینه **دامنه‌ها**، کلیک کنید. در ادامه باید روی گزینه‌ی **افزودن دامنه** کلیک کرده و نام دامنه‌تان (مثلاً: mydomain.com) را وارد کنید. درنهایت با کلیک روی گزینه‌ی **ایجاد دامنه**، دامنه شما به باکت اضافه می‌شود و همچنین به صفحه‌ی مدیریت دامنه هدایت می‌شوید:

![add-domain](https://media.liara.ir/docs/add-domain-to-bucket.gif)

در این مرحله باید رکوردهای DNS درخواست شده را با استفاده از یک سرویس DNS مانند [Cloudflare](https://www.cloudflare.com/) بر روی دامنه‌تان تنظیم کنید.  
درنهایت پس از ثبت رکوردهای DNS می‌توانید بر روی گزینه‌ی  **بررسی وضعیت رکوردها** کلیک کنید و با تغییر تیک‌های قرمز رنگ به سبز، یعنی اتصال دامنه‌ی شما به لیارا با موفقیت صورت گرفته است:

![activated-domain](https://media.liara.ir/docs/activated-domain.png)

در حین ثبت دامنه، نکات زیر را حتماً در نظر بگیرید:
- درصورت استفاده از CDN باید گواهی SSL را نیز از سرویس‌دهنده‌ی DNS فعلی‌تان تهیه کنید.
- درصورت استفاده از CDN می‌توانید تیک قرمز رنگ و خطای تنظیم نشدن رکوردها را نادیده بگیرید.

> شما می‌توانید برای مدیریت دامنه خود، از [سرویس DNS لیارا](https://docs.liara.ir/dns-management-system/about) استفاده کنید تا رکوردها به صورت خودکار به آن اضافه شوند.  
> همچنین بخوانید: [تهیه گواهی SSL برای دامنه](https://docs.liara.ir/paas/domains/enable-ssl)
