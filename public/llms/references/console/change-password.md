# تغییر Password حساب کاربری

برای تغییر Password حساب کاربری خود، کافیست تا بر روی پروفایل خود کلیک کرده و وارد قسمت  
**حساب کاربری** شوید؛  
در این قسمت، باید وارد بخش امنیت شوید؛ و کافیست تا رمز عبور فعلی و رمز عبور جدید خود را وارد کرده، سپس بر روی گزینه تغییر رمز عبور، کلیک کنید تا تغییرات برای‌تان، ذخیره شود:
