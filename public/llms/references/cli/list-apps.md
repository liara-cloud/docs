# مشاهده لیستی از برنامه‌های ایجاد شده با Liara CLI

برای  
مشاهده لیستی از برنامه‌های ایجاد شده با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  
  
```bash
liara app:list
```

## پارامترهای دستور
دستور `liara app:list`یا `liara app:ls`، پارامترهای زیر را، می‌پذیرد:

```bash
liara app ls [-h] [--debug] [--api-token <value>]  [--account <value>] [--columns <value> | -x] [--filter <value>] [--no-header | [--csv | --no-truncate]] [--output csv|json|yaml] [--sort <value>] [--team-id <value>]
```
