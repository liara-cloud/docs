# روشن‌کردن دیتابیس با Liara CLI

برای  
روشن‌کردن دیتابیس با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```bash
liara db start
```

## پارامترهای دستور
دستور `liara db create`، پارامترهای زیر را، می‌پذیرد:

```bash
liara db start [-h] [--debug] [--api-token <value>] 
[--account <value>] [-n <value>] [--team-id <value>]
```
