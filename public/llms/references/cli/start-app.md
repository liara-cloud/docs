# روشن کردن برنامه خاموش با Liara CLI

برای  
روشن کردن برنامه خاموش با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```
liara start
```

## پارامترهای دستور
دستور `liara app:start`یا `liara start`، پارامترهای زیر را، می‌پذیرد:

```
liara start [-h] [--debug] [--api-token <value>]  [--account <value>] [-a <value>] [--team-id <value>]
```
