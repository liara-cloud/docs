# خاموش‌کردن دیتابیس با Liara CLI

برای  
خاموش‌کردن دیتابیس با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```
liara db stop
```

## پارامترهای دستور
دستور `liara db stop`، پارامترهای زیر را، می‌پذیرد:

```
liara db stop [-h] [--debug] [--api-token <value>] 
[--account <value>] [-n <value>] [--team-id <value>]
```
