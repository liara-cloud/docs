# خاموش کردن برنامه با Liara CLI

برای  
خاموش کردن برنامه با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```bash
liara stop
```

## پارامترهای دستور
دستور `liara app:stop`یا `liara stop`، پارامترهای زیر را، می‌پذیرد:

```bash
liara stop [-h] [--debug] [--api-token <value>]  [--account <value>] [-a <value>] [--team-id <value>] [--team-id <value>]
```
