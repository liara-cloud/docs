# تکمیل خودکار دستورات Liara CLI

برای  
تکمیل خودکار دستورات در ابزار Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```bash
liara autocomplete
```

با اجرای این دستور می‌توانید راهنمای فعال‌سازی تکمیل خودکار دستورات لیارا CLI را در سیستم‌عامل فعلی خود مشاهده کنید.
