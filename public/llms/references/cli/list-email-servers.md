# مشاهده لیستی از ایمیل‌سرورهای ایجاد شده با Liara CLI

برای  
مشاهده لیستی از ایمیل‌سرورهای ایجاد شده با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```bash
liara mail ls # or liara mail list
```

## پارامترهای دستور
دستور `liara mail ls`یا `liara mail list`، پارامترهای زیر را، می‌پذیرد:

```bash
 liara mail list [-h] [--debug] [--api-token <value>]  [--account <value>] [--columns
<value> | -x] [--filter <value>] [--no-header | [--csv | --no-truncate]] [--output csv|json|yaml] [--sort
<value>] [--team-id <value>]
```
