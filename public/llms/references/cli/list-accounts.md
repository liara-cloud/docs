# مشاهده لیستی از حساب‌های کاربری با Liara CLI

برای  
مشاهده لیستی از حساب‌های کاربری با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```bash
liara account:list
```

## پارامترهای دستور
دستور `liara account:list`یا `liara account:list`، پارامترهای زیر را، می‌پذیرد:

```bash
 liara account list [-h] [--debug] [--api-token <value>]  [--account <value>] [--columns
<value> | -x] [--filter <value>] [--no-header | [--csv | --no-truncate]] [--output csv|json|yaml] [--sort
<value>]
```
