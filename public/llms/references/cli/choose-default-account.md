# انتخاب حساب پیش‌فرض در Liara CLI

برای  
انتخاب حساب پیش‌فرض در Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```bash
liara account:use
```

## پارامترهای دستور
دستور `liara account:use`، پارامترهای زیر را، می‌پذیرد:

```bash
liara account use [-h] [--debug] [--api-token <value>]  [-a <value>] [--team-id <value>]
```
