# ایجاد سامانه مدیریت دامنه با Liara CLI

برای  
ایجاد سامانه مدیریت برای دامنه مدنظرتان با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```bash
liara zone create
```

## پارامترهای دستور
دستور `liara zone create`، پارامترهای زیر را، می‌پذیرد:

```bash
liara zone create [-h] [--debug] [--api-token <value>] [--account <value>] [-z
<value>] [--team-id <value>]
```
