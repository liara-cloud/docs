# اتصال به خط فرمان برنامه با Liara CLI

برای  
اتصال به خط فرمان برنامه با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:

```bash
liara shell
```

## پارامترهای دستور
دستور `liara app:shell`یا `liara shell`، پارامترهای زیر را، می‌پذیرد:

```bash
liara shell [-h] [--debug] [--api-token <value>]  [--account <value>] [-a <value>] [-c <value>] [--team-id <value>]
```
