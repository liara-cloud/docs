# حذف ایمیل‌سرور با Liara CLI

برای  
حذف ایمیل‌سرور با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:

```bash
liara mail delete
```

## پارامترهای دستور
دستور `liara mail delete` پارامترهای زیر را، می‌پذیرد:

```bash
liara db remove [-h] [--debug] [--api-token <value>] [--account <value>] [-m <value>] [-f] [--team-id <value>]
```
