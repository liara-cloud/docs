# ایجاد ایمیل‌سرور با Liara CLI

برای  
ایجاد ایمیل‌سرور با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:

```bash
liara mail create
```

## پارامترهای دستور
دستور `liara mail create`، پارامترهای زیر را، می‌پذیرد:

```bash
liara mail create [-h] [--debug] [--api-token <value>] [--account <value>] [--domain <value>]
[--plan <value>] [--mode <value>] [--team-id <value>]
```
