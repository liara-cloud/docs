# ایجاد فایل liara.json با Liara CLI

برای  
ایجاد فایل [`liara.json`](https://docs.liara.ir/paas/liarajson/) با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید (دستور را در مسیر اصلی پروژه‌تان اجرا کنید تا فایل  
`liara.json` در مسیر اصلی پروژه‌تان ایجاد شود):

```bash
liara init
```

## پارامترهای دستور
دستور `liara init`، پارامترهای زیر را، می‌پذیرد:

```bash
$ liara init [-h] [--debug] [--api-token <value>] [--account <value>] [-y] [-n <value>] [-p <value>] [-P <value>]
[-v <value>] [--build-location <value>] [-d <value> --path <value>] [--team-id <value>]
```
