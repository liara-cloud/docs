# حذف متغیرهای محیطی با Liara CLI

برای  
حذف متغیرهای محیطی با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```
liara env:unset
```

## پارامترهای دستور
دستور `liara env:unset`، پارامترهای زیر را، می‌پذیرد:

```
liara env unset [ENV...] [-h] [--debug] [--api-token <value>]  [--account <value>] [-a <value>] [-f] [--team-id <value>]
```
