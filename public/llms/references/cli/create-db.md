# ایجاد دیتابیس با Liara CLI

برای  
ایجاد دیتابیس با Liara CLI،  
می‌توانید از دستور زیر، استفاده کنید:  

```bash
liara db create
```

## پارامترهای دستور
دستور `liara db create`، پارامترهای زیر را، می‌پذیرد:

```bash
liara db create [-h] [--debug] [--api-token <value>] [--account <value>] [-n <value>] [--plan <value>] [--public-network] [--feature-plan <value>] [-t <value>] [-v <value>] [-y] [--network <value>] [--team-id <value>]
```
