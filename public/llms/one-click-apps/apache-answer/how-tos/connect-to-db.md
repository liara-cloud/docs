# اتصال به دیتابیس

در قدم دوم نصب برنامه Apache Answer باید تنظیمات مربوط به دیتابیس را پیکربندی کنید. دقت داشته باشید که اگر قصد دارید به دیتابیس در [شبکه خصوصی](https://docs.liara.ir/paas/details/private-network) متصل شوید، برنامه Apache Answer و دیتابیس هر دو باید در یک شبکه خصوصی مشترک قرار داشته باشند. پیشنهاد می‌شود که از دیتابیس SQLite استفاده کنید؛ چون به تنظیمات خاصی احتیاج ندارد و به صورت خودکار، یک [دیسک](https://docs.liara.ir/paas/disks/about) برای انجام این کار در برنامه Apache Answer برای شما، تعریف شده است.
