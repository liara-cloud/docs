# Liara Compose

Liara Compose یک فایل با ماهیت YAML است که می‌تواند شامل مجموعه‌ای از سرویس‌های لیارا نظیر پلتفرم‌ها، برنامه‌های آماده، دیتابیس‌ها و ... باشد که توسط چندین متغیرمحیطی به یکدیگر متصل شده‌اند.

Liara Compose تا حد زیادی مانند [Docker Compose](https://docs.liara.ir/paas/docker/how-tos/deploy-docker-compose) عمل می‌کند و شما می‌توانید از این قابلیت، به عنوان جایگزینی مناسب برای Docker Compose استفاده کنید؛ در ادامه، مستندات استفاده از Liara Compose برای‌تان قرار گرفته است:
