# تنظیم میرورهای پای پی آی

لیارا میرور PyPI را در آدرس زیر، ارائه می‌دهد:

```bash
https://package-mirror.liara.ir/repository/pypi/
```

برای تنظیم میرور فوق کافیست تا دستور زیر را اجرا کنید: 

```bash
export PIP_INDEX_URL=https://package-mirror.liara.ir/repository/pypi/simple
# for windows:
# setx PIP_INDEX_URL "https://package-mirror.liara.ir/repository/pypi/simple"
```

با انجام کار فوق، میرور لیارا برای شما تنظیم می‌شود.
