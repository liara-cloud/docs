# اتصال هوش مصنوعی به برنامه OpenWeb UI

[Video link](https://media.liara.ir/ai/openweb UI-ai.mp4)

برای راه‌اندازی برنامه OpenWeb UI مجهز به هوش مصنوعی، در ابتدا باید، دو سرویس زیر را در لیارا، ایجاد کنید:

- [برنامه آماده OpenWeb](https://docs.liara.ir/one-click-apps/openwebui/quick-start/)
- [سرویس هوش مصنوعی لیارا](https://docs.liara.ir/ai/quick-start/)

بعد از راه‌اندازی دو سرویس فوق، تنها کافیست تا گام‌های زیر را طی کنید: 

## ۱. تغییر متغیرهای محیطی   
وارد بخش [متغیرهای محیطی](https://docs.liara.ir/paas/details/envs/) برنامه OpenWeb UI خود شوید و مقدار `baseUrl` سرویس هوش مصنوعی خود را با مقدار پیش‌فرض متغیر محیطی `OPENAI_API_BASE_URL`، جایگزین کنید.
سپس، مقدار پیش‌فرض متغیر محیطی `OPENAI_API_KEY` را با [کلید API لیارا](https://docs.liara.ir/references/api/about/) جایگزین کنید و سپس بر روی گزینه **ثبت تغییرات** کلیک کنید تا تغییرات ذخیره شوند.

## ۲. ایجاد حساب کاربری جدید
وارد برنامه OpenWeb UI خود شوید و با وارد کردن فیلدهای خواسته شده و کلیک بر روی گزینه `Create Admin Account`، حساب کاربری جدیدی ایجاد کنید.

![create-account-in-openwebui](https://media.liara.ir/ai/openwebui/create-account.png)

با انجام مراحل فوق، سرویس هوش مصنوعی شما در OpenWeb UI آماده استفاده است و می‌توانید پس از انتخاب مدل، از آن استفاده کنید.
