# اتصال هوش مصنوعی به افزونه Kilo Code

افزونه [Kilo Code](https://kilocode.ai) یک ابزار توسعه برای محیط برنامه‌نویسی VS Code است که با هدف تسهیل و تسریع فرآیند کدنویسی طراحی شده است. این افزونه با استفاده از قابلیت‌های هوش مصنوعی، به برنامه‌نویسان کمک می‌کند تا کدهایی با کیفیت بالاتر و در زمان کمتر تولید کنند. از جمله قابلیت‌های آن می‌توان به تکمیل خودکار کد، پیشنهادهای هوشمند، تولید قطعه کدهای آماده (code snippets) و امکان تولید کد بر اساس توضیحات متنی اشاره کرد. Kilo Code برای توسعه‌دهندگان مبتدی و کسانی که زمان محدودی برای نوشتن کد دارند، بسیار مفید است.

برای راه‌اندازی افزونه Kilo Code مجهز به هوش مصنوعی، در ابتدا باید، با کلیک بر روی [این لینک](vscode:extension/kilocode.kilo-code) افزونه Kilo Code را بر روی VSCode خود نصب کنید. پس از نصب آن، بایستی سرویس زیر را در لیارا، ایجاد کنید:

- [سرویس هوش مصنوعی لیارا](https://docs.liara.ir/ai/quick-start/)

بعد از راه‌اندازی، تنها کافیست تا گام‌های زیر را طی کنید: 

[Video link](https://media.liara.ir/ai/kilo-code/setup-kilo-code.mp4)

۱. پیکربندی افزونه در VSCode  
پس از نصب افزونه، از سایدبار، `Kilo Code` را انتخاب کنید و سپس در صفحه باز شده، گزینه `Use your own API Key` را، انتخاب کنید.

۲. اضافه‌کردن هوش مصنوعی لیارا  
در صفحه جدید باز شده، در فیلد **API Provider**، گزینه **OpenAI Compatible** را انتخاب کنید. در قسمت **Base URL**، مقدار baseUrl سرویس هوش مصنوعی خود را وارد کنید و در قسمت **API Key**، بایستی کلید API کنسول لیارا را وارد کنید.  
پس از وارد کردن فیلدهای مذکور، تنها کافیست تا در فیلد **Model**، مدل زبانی (LLM) خود را انتخاب، و در نهایت بر روی گزینه **!Let's go** کلیک کنید.  

با انجام مراحل فوق،  
تمامی کارها انجام شده است و شما، می‌توانید از این افزونه در VSCode، استفاده کنید.
