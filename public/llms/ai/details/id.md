# آیدی (workspace) یک سرویس هوش مصنوعی

آیدی سرویس هوش مصنوعی (workspace) یک شناسه منحصربه‌فرد است که لیارا به هر سرویس هوش مصنوعی که ایجاد می‌کنید، اختصاص می‌دهد. این آیدی برای موارد مختلفی به کار می‌رود. 

## نحوه مشاهده آیدی سرویس هوش مصنوعی
روش‌های مختلفی برای مشاهده آیدی سرویس هوش مصنوعی مدنظرتان، وجود دارد 
که یکی از ساده‌ترین آن‌ها، استفاده از URL کنسول لیارا در حین مشاهده سرویس هوش مصنوعی است.

![see ai id in console](https://media.liara.ir/ai/see-ai-id-in-console.png)

آیدی سرویس هوش مصنوعی بعد از `ai/` و قبل از `/` بعدی، قرار دارد.
به‌عنوان مثال، اگر URL کنسول شما در صفحه **مدل‌ها** سرویس هوش مصنوعی‌تان مانند زیر باشد:

```bash
https://console.liara.ir/ai/682b7ab82bd70203933013ae/models
```

بنابراین آیدی سرویس هوش مصنوعی شما برابر با `682b7ab82bd70203933013ae` است.
